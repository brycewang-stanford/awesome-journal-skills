# QJE Pre-Submission Self-Check

A self-check for manuscripts targeted at the *Quarterly Journal of Economics* (QJE),
submitted via Editorial Express. Verify all volatile specifics (fee, deposit policy,
submission URL, current anonymization rules) on the official journal page.

## 1. Fit & framing

- [ ] One-sentence hook: surprising answer + clean source of variation
- [ ] The question is legible to a general-interest (non-specialist) top-5 reader
- [ ] The "broad lesson beyond this setting" is explicit
- [ ] The paper is not better suited to Econometrica (method) or a top field journal (incremental)

## 2. Contribution & literature

- [ ] Contribution stated in the introduction, not a standalone survey section
- [ ] The 3–6 closest frontier papers cited precisely and characterized fairly
- [ ] The delta (what is new) stated in one or two crisp sentences
- [ ] Likely-referee papers engaged, not ignored

## 3. Identification

- [ ] Identifying variation named in one sentence and defended as exogenous
- [ ] Design-appropriate diagnostics done (pre-trends / density / first-stage F / balance)
- [ ] Modern estimator used for staggered DID (Callaway–Sant'Anna / Sun–Abraham / dCDH); Goodman-Bacon reported
- [ ] Inference matched to the assignment level; few-cluster issues handled
- [ ] Placebo / falsification evidence reported
- [ ] LATE / external-validity interpretation stated

## 4. Conceptual frame

- [ ] The estimated object mapped to an economic primitive
- [ ] A testable prediction the data address (verbal frame or model)
- [ ] Theory weight matched to the question (no gratuitous structural model)
- [ ] Welfare / policy interpretation offered where feasible

## 5. Robustness & online appendix

- [ ] Every identification threat has a corresponding, named check
- [ ] Estimate shown insensitive to reasonable specification choices
- [ ] Appendix organized by threat category, each result summarized in the body
- [ ] Magnitudes benchmarked against prior estimates

## 6. Exhibits

- [ ] Central result has a headline figure (event study / RDD plot / binscatter / first stage)
- [ ] Main table readable; sprawling variants moved to the appendix
- [ ] Every exhibit note self-contained (sample, units, FE, clustering, significance)
- [ ] Magnitudes interpretable (units stated)

## 7. Writing & references

- [ ] Abstract states the actual finding with a number
- [ ] Introduction follows the arc: question → hardness → variation → result → interpretation → contribution
- [ ] Author-date references throughout, consistent and complete
- [ ] Jargon minimized for a general-interest reader; no over-claiming

## 8. Submission system & files

- [ ] Submitting through Editorial Express (confirm current URL)
- [ ] Submission fee verified (current amount) and payment ready
- [ ] Manuscript PDF + separate online appendix + concise cover letter staged
- [ ] Anonymized per current policy (body + file metadata); acknowledgments/funding separate
- [ ] Suggested / excluded referees prepared (expert, fair, conflict-free)
- [ ] Conflict-of-interest, funding, and data disclosures prepared
- [ ] Replication materials (data + code) staged for the accepted stage
- [ ] Paper not under review elsewhere
