# 《经济纵横》投稿（economic-aspects）

## 期刊定位

综合经济理论与政策刊，适合宏观形势、改革、产业政策和现实经济问题的学术化分析。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《经济纵横》或 Economic Review Journal
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《经济纵横》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 选题偏好

- 宏观经济与政策形势
- 改革开放、产业发展和区域经济
- 共同富裕、数字经济和绿色转型

## 方法与证据

- 适合经济体制改革、制度比较、区域试点、营商环境和政策评估
- 经验研究要把政策背景、试点选择和反事实讲清
- 规范分析要有制度逻辑和可验证材料支撑

## 结构与写作

- 引言要围绕改革问题的制度瓶颈展开
- 贡献强调制度机制、改革路径和中国经验的一般化含义
- 政策建议应具体但有理论支撑

## 官方核验清单

- 核对栏目、字数、摘要关键词、作者匿名和参考文献要求
- 确认政策文件、试点名单和改革时间线
- 检索“《经济纵横》 投稿须知 / 征稿简则 / 作者指南”，以最新页面为准；不要沿用二手代投网站格式
- 核对是否要求匿名正文、作者信息页、基金信息单列、数据/程序/补充材料、版权协议或不收费声明
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《经济纵横》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 方法、数据、图表和参考文献格式已按最新官方稿约复核
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 写成政策宣传或经验总结，缺学术问题
- 改革概念过大，变量和机制不可操作
- 只比较政策文件，不检验证据

## 改投判断

严格实证转 economic-review-cn；制度比较转 comparative-economic-and-social-systems。

## 输出格式

```text

【目标期刊】《经济纵横》
【选题标签】<最贴近的 2-3 个主题>
【方法证据】<当前方法是否够本刊标准>
【最大风险】<最可能导致退稿的一点>
【需核验官方要求】<投稿系统/匿名/摘要/图表/数据等>
【改投建议】<若不匹配，给出更合适期刊>
```
