# 《管理科学学报》投稿（journal-of-management-sciences-china）

## 期刊定位

国家自然科学基金委管理学部认定的**数理/定量管理科学**顶刊。它与《管理世界》是两条路：《管理世界》偏经验实证与案例，本刊偏**模型、证明、算法、优化**。核心标识是**数理严谨性**——运筹优化、决策理论、数理金融、行为运作、系统建模、供应链与博弈。一篇问卷-SEM 或纯案例文，多半不对口。

## 触发时机

- 研究产出是**模型 + 定理/命题 + 算法/数值实验**
- 题目涉及运筹优化、排队、库存、定价、博弈、机制设计、数理金融、风险度量
- 行为运作（behavioral OM）：实验 + 模型刻画偏差
- 不确定该投本刊还是《管理世界》（经验型）

## 选题偏好

1. **优化与决策**：供应链、库存、定价、调度、网络优化、鲁棒/随机优化
2. **博弈与机制设计**：契约设计、竞争与合作、平台与定价博弈
3. **数理金融与风险**：投资组合、风险度量、衍生品、最优控制
4. **行为运作与实验**：用实验数据校准/检验行为模型
5. **数据驱动决策**：运筹 + 机器学习的方法贡献

## 方法与规范（数理为核）

- 必须有**清晰的模型设定**：决策变量、约束、目标函数、假设
- **命题/定理需证明**（正文或附录），逻辑链完整
- 算法要给出复杂度/收敛性论证，配数值实验或仿真验证
- 行为/实验类：实验设计规范，统计检验 + 模型识别并重
- 评判重点是**方法贡献**（新模型/新算法/新性质），而非政策含义

## 结构与写作

- 引言：问题动机 → 文献中方法缺口 → 本文模型与方法贡献
- 模型—分析—算法/数值—管理启示，结构紧凑
- 记号统一、假设显式、定理表述严谨；证明放附录保持正文流畅
- 数值实验要回答"模型/算法在什么条件下更优"，给管理洞见
- 管理启示是"模型告诉我们什么决策规律"，不是政策口号

## 官方核验清单

- 检索“《管理科学学报》 投稿须知 / 征稿简则 / 作者指南”，以官网、主办单位页面、采编系统或期刊官方 CNKI 页面为准
- 核对是否要求匿名正文、作者信息页、基金信息单列、摘要/关键词/JEL 或中图分类号、图表/参考文献格式
- 核对是否有数据、程序、补充材料、版权协议、不收费声明或特殊栏目要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 模型设定完整（变量/约束/目标/假设显式）
- [ ] 关键结论以命题/定理给出并证明
- [ ] 算法有复杂度或收敛性分析
- [ ] 数值/仿真验证了理论性质并给出管理洞见
- [ ] 记号与表述前后一致、严谨
- [ ] 贡献定位在方法层（新模型/性质/算法），非政策

## 高频拒稿雷区

- 把问卷-SEM 或纯案例投到数理刊（学科不匹配）
- 有模型无证明，或命题表述含糊
- 算法无复杂度/收敛分析，数值实验单薄
- 假设不合理或与建模目标脱节
- 用"管理意义"套话替代方法贡献

## 决策口诀

- "产出是定理/算法" → 本刊对口
- "产出是回归系数 + 政策含义" → 《管理世界》/《中国工业经济》
- "产出是组织行为理论 + 问卷" → 《南开管理评论》
- "数理金融偏资产定价实证" → 也可考虑《金融研究》

## 输出格式

```
【匹配度】高 / 中 / 低
【产出类型】模型+定理 / 算法 / 行为实验+模型 / 仅实证（不匹配）
【严谨性】假设显式 □ 证明完整 □ 复杂度/收敛 □
【数值验证】充分 / 单薄
【方法贡献】<一句话：新在何处>
【建议去向】管理科学学报 / 管理世界 / 南开管理评论
```
