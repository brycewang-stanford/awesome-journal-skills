# 《社会保障评论》投稿（social-security-studies）

## 期刊定位

社会保障专业刊，适合养老、医保、社保基金、社会救助和共同富裕研究。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《社会保障评论》或 Social Security Studies
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《社会保障评论》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 选题偏好

- 养老保险、医保和长期护理保险
- 社保基金、缴费激励和财政可持续
- 社会救助、就业保障和收入分配

## 方法与证据

- 社会保障、养老、医保、收入分配、公共服务和社会政策评估是核心
- 可用微观调查、行政数据、政策试点、精算模型、DID/IV/RDD 或制度比较
- 机制要落到保障激励、风险分担、家庭决策和财政可持续

## 结构与写作

- 引言同时交代制度背景、政策问题和学术争论
- 结果解释要兼顾公平、效率和可持续
- 政策含义面向制度设计和执行层级

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 核对调查/行政数据授权、伦理、隐私和投稿格式
- 确认政策文件、保险口径和样本权重
- 检索“《社会保障评论》 投稿须知 / 征稿简则 / 作者指南”，以最新页面为准；不要沿用二手代投网站格式
- 核对是否要求匿名正文、作者信息页、基金信息单列、数据/程序/补充材料、版权协议或不收费声明
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《社会保障评论》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 方法、数据、图表和参考文献格式已按最新官方稿约复核
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 只描述政策条款，不检验证据
- 忽略财政可持续和群体异质性
- 把社会政策写成一般宏观经济题

## 改投判断

劳动经济机制转 studies-in-labor-economics；公共政策综合转 public-administration-and-policy-review。

## 输出格式

```text

【目标期刊】《社会保障评论》
【选题标签】<最贴近的 2-3 个主题>
【方法证据】<当前方法是否够本刊标准>
【最大风险】<最可能导致退稿的一点>
【需核验官方要求】<投稿系统/匿名/摘要/图表/数据等>
【改投建议】<若不匹配，给出更合适期刊>
```
