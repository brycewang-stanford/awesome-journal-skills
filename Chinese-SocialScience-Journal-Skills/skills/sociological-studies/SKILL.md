# 《社会学研究》投稿（sociological-studies）

## 期刊定位

中国社会科学院社会学研究所主办，国内**社会学第一刊**。它接纳**定量**（大型社会调查的统计分析）与**定性**（田野/民族志/深度访谈/历史比较）两条传统，但共同要求是**与社会学理论的实质对话**。核心不是"识别干净"，而是**社会学问题意识 + 理论想象力 + 经验扎实**。经济学式的"政策评估"框架在这里会显得问题意识缺位。

## 触发时机

- 题目涉及社会分层与流动、家庭与性别、社会网络与资本、城乡与户籍、劳动与组织、社会治理
- 数据是 CGSS / CFPS / CLDS / 人口普查等社会调查，或自采田野/访谈材料
- 文章像"经济学实证"，缺社会学理论对话与问题意识
- 不确定定量稿与定性稿如何分别写到位

## 选题偏好

1. **社会分层与不平等**：教育、职业、收入、代际流动、机会结构
2. **家庭、性别与生命历程**：婚育、家庭策略、性别分工
3. **社会网络与社会资本**：关系、信任、组织与集体行动
4. **城乡、流动与治理**：户籍、迁移、社区、基层治理、数字社会

## 方法（定量 / 定性并重）

- **定量**：回归、Logit/多层模型、事件史/生存分析、序列分析、结构方程；强调**机制与异质性**的社会学解释，而非因果识别军备竞赛
- **定性**：田野民族志、深度访谈、扩展个案、历史比较；强调材料厚度、分析深度、理论提炼与可信度（饱和、反身性）
- 混合方法可加分，但每种方法都要做规范
- 描述性事实若揭示重要社会结构，本刊比经济学刊更接纳

## 结构与写作

- 引言立**社会学问题意识**：这是关于何种社会机制/结构的问题，与哪个理论对话
- 文献回顾要进入**理论脉络**（经典与当代社会学），不是变量清单
- 定量稿：把系数翻译成**社会过程**（谁、经由什么机制、形成什么结构）
- 定性稿：从材料中**提炼概念/机制**，与理论往返
- 文风学理、节制，避免政策汇报腔与纯技术腔

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 检索“《社会学研究》 投稿须知 / 征稿简则 / 作者指南”，以官网、主办单位页面、采编系统或期刊官方 CNKI 页面为准
- 核对是否要求匿名正文、作者信息页、基金信息单列、摘要/关键词/JEL 或中图分类号、图表/参考文献格式
- 核对是否有数据、程序、补充材料、版权协议、不收费声明或特殊栏目要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有清晰的社会学问题意识，而非"X 对 Y 的影响"
- [ ] 与社会学理论脉络有实质对话
- [ ] 定量：机制/异质性有社会学解释，不止报系数
- [ ] 定性：材料厚、编码透明、概念提炼到位
- [ ] 结论回扣并推进了社会学理论
- [ ] 无政策汇报腔、无纯方法炫技

## 高频拒稿雷区

- 经济学式政策评估，社会学问题意识缺位
- 文献只列变量关系，不进理论脉络
- 定量稿把显著性当结论，无社会过程解释
- 定性稿停在故事复述，无概念提炼
- 把方法（新数据/新模型）当贡献本身

## 决策口诀

- "社会结构/机制 + 理论对话" → 本刊对口
- "落点是经济因果 + 政策含义" → 《经济研究》/《中国工业经济》
- "跨学科大理论命题" → 可冲《中国社会科学》
- "组织内构念关系 + 问卷" → 偏《南开管理评论》

## 输出格式

```
【匹配度】高 / 中 / 低
【路径】定量(数据源) / 定性(田野·访谈·历史) / 混合
【问题意识】<社会学问题一句话>
【理论对话】<经典/当代理论> | 缺失
【机制解释】社会过程清晰 / 仅系数（需补）
【贡献】推进/修正 <理论>
```
