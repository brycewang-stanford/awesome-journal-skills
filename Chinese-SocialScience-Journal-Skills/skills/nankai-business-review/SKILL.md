# 《南开管理评论》投稿（nankai-business-review）

## 期刊定位

国内**商学/组织管理**顶刊。学科底色是**管理学理论建构**：战略管理、组织行为、人力资源、市场营销、公司治理、创业。与《管理世界》相比更"学院派管理学"，与《管理科学学报》相比更"行为/理论"而非"数理优化"。核心评判：**理论贡献 + 中国情境 + 方法规范（问卷实证 / 多案例）。**

## 触发时机

- 题目是组织/战略/营销/治理的**构念间关系**（中介、调节、被调节的中介）
- 数据是**问卷/量表**（个体或企业层级），方法是 SEM / 多层线性模型 / 实验
- 或是**多案例 / 扎根理论**的质性建构
- 想判断该投《南开管理评论》《管理世界》还是《管理科学学报》

## 选题偏好

1. **组织行为与人力资源**：领导、团队、员工态度行为、HR 实践
2. **战略与创业**：战略选择、动态能力、商业模式、创业过程
3. **市场营销**：消费者行为、品牌、渠道、数字营销
4. **公司治理**：董事会、股权、高管激励（偏管理学视角而非纯财务）

## 方法与规范

- **问卷实证**：量表需有信度（Cronbach's α、CR）、效度（收敛/区分效度、AVE）、共同方法偏差检验（Harman/标记变量）
- 模型多为**中介/调节/被调节的中介**（Bootstrap 检验间接效应）；多层数据用 HLM
- 实验法日益受欢迎：操纵检验、随机化、效应量
- **质性研究**：多案例比较、扎根理论编码（开放-主轴-选择），讲清理论饱和与可信度
- 重点不是因果识别的"干净"，而是**构念清晰 + 理论逻辑 + 测量规范**

## 结构与写作

- 引言立**理论缺口**：现有理论解释不了什么，本文补什么命题
- 假设推演：每条假设有理论机理支撑（而非"我觉得正相关"）
- 讨论部分要回扣理论贡献（对哪个理论的扩展/边界条件/整合）
- 中国情境是加分项：情境如何改变构念关系，提炼情境化理论

## 官方核验清单

- 检索“《南开管理评论》 投稿须知 / 征稿简则 / 作者指南”，以官网、主办单位页面、采编系统或期刊官方 CNKI 页面为准
- 核对是否要求匿名正文、作者信息页、基金信息单列、摘要/关键词/JEL 或中图分类号、图表/参考文献格式
- 核对是否有数据、程序、补充材料、版权协议、不收费声明或特殊栏目要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 有清晰的理论缺口与理论贡献陈述
- [ ] 每条假设有理论机理，不是经验直觉
- [ ] 量表信效度 + 共同方法偏差检验齐全
- [ ] 中介/调节用 Bootstrap，多层数据用合适模型
- [ ] 质性研究编码过程透明、可信度论证充分
- [ ] 讨论回扣并推进了某个管理理论

## 高频拒稿雷区

- 假设全凭直觉，无理论机理（"应该正相关"）
- 量表照搬国外未做信效度/CMV 检验
- 把回归显著当理论贡献，讨论部分无理论对话
- 中国情境只是采样地，没进入理论
- 用经济学政策评估的框架投管理学刊（错位）

## 决策口诀

- "构念关系 + 问卷/案例 + 理论建构" → 本刊对口
- "宏观政策实证 + 政策含义" → 《管理世界》/《中国工业经济》
- "数理模型 + 定理/算法" → 《管理科学学报》
- "公司治理偏资本市场实证" → 可考虑《会计研究》/《金融研究》

## 输出格式

```
【匹配度】高 / 中 / 低
【方法】问卷-SEM / 实验 / 多案例-扎根 / 其他
【理论缺口】<一句话>
【测量规范】信度 □ 效度 □ CMV □（问卷类）
【理论贡献】扩展 / 边界条件 / 整合 <理论名>
【情境化】中国情境是否进入理论：是 / 否
```
