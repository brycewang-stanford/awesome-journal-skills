# 《数量经济技术经济研究》投稿（journal-of-quantitative-technological-economics）

## 期刊定位

中国社会科学院数量经济与技术经济研究所主办。两条主线：**数量经济学**（计量方法及其应用、宏观计量、经济预测、投入产出、CGE）与**技术经济学**（技术进步、生产率、效率评价、创新与数字经济的测度与评估）。相较《经济研究》《CEQ》偏理论/因果，本刊更看重**测度与方法的应用价值**——把"怎么量、量得准、预测得好"做扎实。

## 触发时机

- 贡献点在**测度/方法**：全要素生产率、效率（DEA/SFA）、技术进步分解、投入产出/CGE、指数构建
- 题目是宏观计量、经济预测、政策模拟、新经济（数字经济/AI）测度
- 文章方法应用扎实但"因果故事"不强，担心被纯因果刊嫌识别弱
- 想判断该投本刊还是《经济研究》/《中国工业经济》

## 选题偏好

1. **生产率与效率测度**：TFP、Malmquist、DEA/SFA、技术进步分解
2. **宏观计量与预测**：时间序列、混频/大数据计量、景气与预测、政策模拟
3. **投入产出与一般均衡**：IO 表分析、CGE、结构分解（SDA）
4. **技术经济评价**：技术/项目经济评价、创新与数字经济的量化
5. **计量方法的应用与改进**：把前沿方法用到中国数据并讲清适用条件

## 方法与规范（测度/方法为核）

- 测度类：方法选择有依据，**指标构造透明**，对测度敏感性（不同方法/参数）做检验
- 计量/预测类：模型设定、平稳性/协整、样本外预测评估（RMSE/方向准确率）规范
- IO/CGE：数据来源、参数校准、闭合规则、情景设定清楚，结果可复现
- 因果识别不是必需，但若做因果，仍按现代规范（DID 异质性、IV 推断）
- 评判重点：**测得准不准、方法用得对不对、结论稳不稳**

## 结构与写作

- 引言：现实/政策需求 → 现有测度或方法的不足 → 本文方法贡献
- 方法节是重头：把模型/算法/指标讲透，假设与适用条件显式
- 结果要给**量化解读 + 敏感性**，技术经济类要落到评价结论
- 政策/实务含义面向规划、预测、产业与技术决策

## 投稿前自检

- [ ] 贡献点清楚是"测度/方法应用"还是"因果"，并据此组织全文
- [ ] 指标/模型构造透明、可复现
- [ ] 做了测度/参数敏感性或样本外评估
- [ ] 数据来源与口径准确（宏观/IO/行业数据）
- [ ] 方法适用条件与局限有交代
- [ ] 结论给出量化解读，不止显著性

## 高频拒稿雷区

- 用现成软件跑 DEA/SFA/CGE 却不交代设定与参数来源
- 预测模型不做样本外评估，只报样本内拟合
- 指数/测度构造不透明，无法复现
- 强行包装因果故事但识别站不住（不如老实做测度）
- 方法堆砌，缺对中国数据/政策的实际意义

## 决策口诀

- "贡献在测度/预测/方法应用" → 本刊对口
- "贡献在干净因果 + 理论" → 《经济研究》/《CEQ》
- "贡献在产业政策评估 + 厚稳健性" → 《中国工业经济》
- "贡献在数理模型/算法本身" → 《管理科学学报》

## 输出格式

```
【匹配度】高 / 中 / 低
【贡献类型】测度 / 预测 / IO-CGE / 计量方法应用 / 因果
【方法透明度】构造可复现 □ 敏感性/样本外评估 □
【数据口径】准确 / 存疑 <点>
【实务含义】规划 / 预测 / 产业 / 技术决策
【建议去向】JQTE / 经济研究 / 中国工业经济 / 管理科学学报
```
