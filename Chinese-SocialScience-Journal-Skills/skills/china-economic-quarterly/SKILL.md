# 《经济学(季刊)》投稿（china-economic-quarterly）

## 期刊定位

北京大学中国经济研究中心（CCER）主办，是国内**最贴近国际主流经济学范式**的中文刊。编委与审稿人多有海外训练，看重"问题—模型/识别—证据"的逻辑闭环。可发表中文或英文。与《经济研究》相比，CEQ 更强调**方法规范性与技术严谨**，对政策口号的容忍度更低。

## 触发时机

- 文章方法接近国际期刊范式，想找国内最对口的中文出口
- 有干净的准实验设计或结构模型，但写作还停在"国内实证套路"
- 担心识别策略 / 标准误 / 推断细节经不起技术审稿
- 想判断该投 CEQ 还是《经济研究》

## 选题偏好

1. **机制清楚的因果问题**：理论上可建模、经验上可识别
2. **方法本身有规范价值**：结构估计、新识别策略、计量应用的范本
3. **国际可对话**：问题表述能让 field 同行立刻看懂贡献，不依赖本土语境包装
4. 不偏好纯描述、纯政策评估、缺识别的相关性研究

## 方法与识别（最高权重）

- **要么结构、要么准实验**：清晰的结构模型 + 可信参数，或干净的 DID/IV/RDD/event-study
- 双向固定效应 DID 必须回应**异质性处理效应**批评（Callaway–Sant'Anna / de Chaisemartin–D'Haultfœuille / Sun–Abraham），不能只放 TWFE
- IV 要论证**外生性与排他性**，报告第一阶段 F、弱工具稳健推断
- 标准误聚类层级要说明理由；必要时多重假设检验校正
- 安慰剂、平行趋势、敏感性分析是**标配而非加分项**

## 结构与写作

- 引言：问题 → 为何难 → 本文怎么解决（识别/模型）→ 主要发现 → 与文献的精确定位
- 贡献写作**对标 field**：与具体文献的方法/结论差异，而非"丰富了研究"
- 理论部分要么给可检验命题，要么给结构方程；不要"文献综述式理论"
- 图优于表：event-study 图、bin-scatter、模型拟合图受欢迎
- 全英文摘要质量要高（很多读者只读英文摘要 + 图）

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 检索“《经济学(季刊)》 投稿须知 / 征稿简则 / 作者指南”，以官网、主办单位页面、采编系统或期刊官方 CNKI 页面为准
- 核对是否要求匿名正文、作者信息页、基金信息单列、摘要/关键词/JEL 或中图分类号、图表/参考文献格式
- 核对是否有数据、程序、补充材料、版权协议、不收费声明或特殊栏目要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 识别假设被显式写出，并逐条提供经验支持
- [ ] DID 已处理交错处理/异质性效应的现代批评
- [ ] 推断细节（聚类、弱工具、多重检验）经得起技术审稿
- [ ] 贡献定位精确到具体文献，不是泛化套话
- [ ] 图能独立讲清主结果
- [ ] 英文摘要与术语规范、可被 field 同行读懂

## 高频拒稿雷区

- OLS + 控制变量当因果，未讨论内生性
- TWFE-DID 不回应近年异质性处理效应文献
- 政策腔贡献（"为政府提供参考"）——CEQ 尤其反感
- 理论部分是文献堆砌，没有可检验命题或模型
- 稳健性章节缺平行趋势/安慰剂

## 决策口诀

- "识别干净 + 国际可对话" → CEQ 正中靶心
- "理论贡献厚但识别一般" → 可能更适合《经济研究》
- "纯政策评估、无机制" → 改投行业刊或补识别
- "我只有相关性" → 先解决因果，否则别投

## 输出格式

```
【匹配度】高 / 中 / 低
【识别策略】结构 / 准实验(类型) / 仅相关（需补）
【现代DID合规】是 / 否（缺哪一步）
【推断风险】<聚类/弱工具/多重检验 待补项>
【贡献定位】对标到 <具体文献> | 仍是套话（需改）
【建议去向】CEQ / 经济研究 / 行业刊
```
