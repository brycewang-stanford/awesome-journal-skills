# 《经济问题探索》投稿（inquiry-into-economic-issues）

## 期刊定位

云南财经/区域经济特色明显，适合区域发展、开放边疆、产业和政策探索。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《经济问题探索》或 Inquiry into Economic Issues
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《经济问题探索》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 选题偏好

- 西南开放、边疆经济和区域协同
- 产业升级、数字经济和绿色发展
- 财政金融、乡村振兴和共同富裕

## 方法与证据

- 区域经济、产业经济、财政金融、企业管理和开放经济均可，关键是问题清楚、证据扎实、体例规范
- 高校学报通常要求规范写作、完整中英文摘要关键词、基金信息和参考文献格式
- 经验研究仍需处理内生性和稳健性，但可比顶刊更强调区域特色和政策解释

## 结构与写作

- 引言把区域或学校学科特色转化为明确学术问题
- 文献综述避免罗列，必须说明本文位置
- 结论给出适度政策启示和研究边界

## 官方核验清单

- 核对官网稿约、匿名/非匿名、基金脚注、摘要关键词和参考文献格式
- 确认是否接受网络首发和数据库收录声明
- 检索“《经济问题探索》 投稿须知 / 征稿简则 / 作者指南”，以最新页面为准；不要沿用二手代投网站格式
- 核对是否要求匿名正文、作者信息页、基金信息单列、数据/程序/补充材料、版权协议或不收费声明
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《经济问题探索》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 方法、数据、图表和参考文献格式已按最新官方稿约复核
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 像地方工作报告或泛泛综述
- 数据口径和变量定义不清
- 未按学报体例处理作者信息、基金和参考文献

## 改投判断

国际经贸更专业转 international-economics-and-trade-research；区域综合转 south-china-journal-of-economics。

## 输出格式

```text
【匹配度】高 / 中 / 低（一句话理由）
【目标期刊】《经济问题探索》
【选题标签】<最贴近的 2-3 个主题>
【方法证据】<当前方法是否够本刊标准>
【最大风险】<最可能导致退稿的一点>
【需核验官方要求】<投稿系统/匿名/摘要/图表/数据等>
【改投建议】<若不匹配，给出更合适期刊>
```
