# 《世界经济》投稿（journal-of-world-economy）

## 期刊定位

中国社会科学院世界经济与政治研究所主办，国内**国际经济学/开放宏观**第一刊。核心标识是**国际视角**：贸易、投资、金融、汇率、全球价值链、开放与增长。一篇纯国内、与"开放/外部"无关的文章，即便做得好，也属于不对口。

## 触发时机

- 题目涉及贸易、关税、FDI、汇率、跨境资本、GVC、全球冲击
- 用中国开放政策（入世、自贸区、一带一路、外资准入）做准实验
- 文章有国际维度但写成了"国内政策评估"，定位偏了
- 想判断该投《世界经济》还是《金融研究》/《中国工业经济》

## 选题偏好

1. **开放经济中的因果问题**：外部冲击/政策如何通过贸易、投资、金融渠道传导
2. **国际比较与全球视野**：跨国数据、多国面板、全球价值链分解
3. **中国与世界的互动**：中国开放进程对内/对外的双向影响
4. **国际经济理论的中国检验或扩展**

## 方法与数据

- 主流：贸易/海关微观数据（如企业-产品-目的国）、跨国宏观面板、投入产出/GVC 分解（如 KWW、ADB MRIO）
- 准实验识别同样被看重：关税冲击、贸易协定、汇改、外资负面清单调整作为外生变动
- 引力模型、event-study、shift-share（Bartik）工具变量常见——需注意 shift-share 推断的现代批评
- 宏观金融题可用 VAR/局部投影刻画冲击传导

## 结构与写作

- 引言要把"国际/开放"这一维度立在最前，让审稿人一眼看出对口
- 机制要讲清**传导渠道**（贸易渠道、金融渠道、预期渠道……）
- 与国际贸易/国际金融经典与前沿对话（不能只引国内文献）
- 政策含义偏**开放政策**层面（贸易、汇率、资本账户、产业开放）

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 检索“《世界经济》 投稿须知 / 征稿简则 / 作者指南”，以官网、主办单位页面、采编系统或期刊官方 CNKI 页面为准
- 核对是否要求匿名正文、作者信息页、基金信息单列、摘要/关键词/JEL 或中图分类号、图表/参考文献格式
- 核对是否有数据、程序、补充材料、版权协议、不收费声明或特殊栏目要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] "国际/开放"维度在引言前段就立住，不是硬贴标签
- [ ] 传导渠道明确（经由贸易/投资/金融/预期）
- [ ] 数据匹配国际经济研究（海关/跨国面板/GVC）
- [ ] shift-share / 引力模型的识别与推断站得住
- [ ] 文献含国际贸易/金融的经典与前沿
- [ ] 政策含义落在开放政策层面

## 高频拒稿雷区

- 题目其实是国内问题，"国际"只是引言里一句话
- 用了 GVC 词汇但没做分解，只是叙事
- shift-share IV 不讨论 exposure 外生性/推断
- 只引国内文献，缺国际对话
- 机制停在"促进了出口"，没说清渠道

## 决策口诀

- "外部冲击/开放政策 + 传导渠道" → 《世界经济》对口
- "落点在银行/货币政策/资本市场" → 更适合《金融研究》
- "落点在国内产业/企业行为" → 更适合《中国工业经济》
- "跨国比较的纯增长理论" → 可冲本刊，但需经验支撑

## 输出格式

```
【匹配度】高 / 中 / 低
【国际维度】贸易 / 投资 / 金融 / 汇率 / GVC | 偏弱（需上移）
【传导渠道】<明确的渠道> | 缺失
【识别策略】<外生冲击/IV/event-study>
【数据对口度】海关/跨国/GVC | 一般国内数据
【建议去向】世界经济 / 金融研究 / 中国工业经济
```
