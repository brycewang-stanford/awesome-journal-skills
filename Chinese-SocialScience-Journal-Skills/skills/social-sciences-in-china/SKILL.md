# 《中国社会科学》投稿（social-sciences-in-china）

## 期刊定位

中国社会科学院主办，综合性、跨学科社科第一刊。它不是"经济学刊"也不是"管理学刊"——经济、社会、政治、法学、历史、哲学共处一刊。**衡量标准是思想分量，而非方法复杂度。** 一篇技术精致但问题琐碎的实证文，几乎必然不在射程内。

## 触发时机

- 想冲国内社科最高平台，需要先判断"够不够格"
- 手头有大数据 / 好识别，但讲不出"大问题"
- 文章学科归属模糊，想知道往哪个学科叙事靠
- 通篇是"估计系数 + 稳健性"，缺少理论命题

## 选题偏好（按优先级）

1. **重大理论与现实问题**：能与国家发展、文明、制度、治理这一层级对话
2. **原创性理论贡献**：提出概念、命题或分析框架，而非验证已有结论
3. **中国自主知识体系**：从中国经验提炼一般性理论，回应"中国之问/时代之问"
4. **学科交叉**：单一学科的标准技术活，不如有理论纵深的跨学科叙事

> 黄金句式：本文不只回答"X 是否影响 Y"，而是回答"X→Y 这一现象揭示了什么一般机理 / 修正了哪个既有理论判断"。

## 方法态度：方法服务于论题

- 实证、思辨、规范、历史比较——**方法多元，不偏好某一种**
- 计量只是证据之一，**不能替代理论论证**；纯回归文需要补足理论命题层
- 用了高级方法，必须翻译成"它帮我们看清了什么思想问题"，否则就是炫技
- 概念界定、逻辑推演的严谨度，权重不低于实证识别

## 结构与写作

- 引言先立"问题的重要性与争论焦点"，再立本文命题，最后才谈数据/方法
- 全文围绕**核心命题**展开，每一节回答"这一步如何支撑命题"
- 文风：宏观视野 + 学理克制，**忌口号化、忌政策汇报腔**
- 文献对话要触及**理论源流**（经典作者与流派），不能只堆近五年实证
- 篇幅通常较长（2 万字以上常见），结构完整，注释规范（脚注/页下注体例）

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 检索“《中国社会科学》 投稿须知 / 征稿简则 / 作者指南”，以官网、主办单位页面、采编系统或期刊官方 CNKI 页面为准
- 核对是否要求匿名正文、作者信息页、基金信息单列、摘要/关键词/JEL 或中图分类号、图表/参考文献格式
- 核对是否有数据、程序、补充材料、版权协议、不收费声明或特殊栏目要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 能用一句话说清"本文的原创理论命题"，而非"本文的实证发现"
- [ ] 问题的重要性立在国家/文明/制度/治理层级，不是某细分市场
- [ ] 有与经典理论流派的实质对话，不是文献罗列
- [ ] 方法被翻译成了思想发现，没有停在系数表
- [ ] 没有政策汇报腔（"加强、完善、推进"四件套出现 0 次）
- [ ] 体例符合该刊脚注/参考文献规范

## 高频不匹配信号（≈ desk reject）

- 选题是"某政策对某结果的评估"，没有上升到理论命题
- 通篇技术展示，理论贡献只有结尾一段套话
- 学科叙事摇摆，既不像经济学也不像社会学，谁都不认领
- 把"数据新/方法新"当成贡献本身

## 决策口诀

- "我有发现但没命题" → 先补理论命题层，否则别投
- "我能讲清大问题但证据弱" → 仍比"证据强但无问题"更接近射程
- "这其实是篇好的产业经济实证" → 改投《中国工业经济》或《经济研究》
- "这其实是篇定量社会学" → 仍可冲本刊，但要把社会学理论对话做足，否则退守《社会学研究》

## 输出格式

```
【匹配度】高 / 中 / 低（理由一句话）
【原创命题】<本文可一句话陈述的理论命题>
【问题层级】文明/制度/治理 | 行业/细分（后者需上移）
【理论对话缺口】<缺哪个流派/经典>
【炫技风险】有 / 无（方法是否翻译成思想）
【建议去向】本刊 / 改投 <期刊>
```
