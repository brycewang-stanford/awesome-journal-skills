# 《亚太经济》投稿（asia-pacific-economic-review）

## 期刊定位

亚太区域经济刊，适合区域合作、国别经济、贸易投资和地缘经济研究。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《亚太经济》或 Asia-Pacific Economic Review
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《亚太经济》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 选题偏好

- 亚太区域合作、RCEP 和产业链
- 国别经济、贸易投资和跨境金融
- 中国与亚太经济互动和开放战略

## 方法与证据

- 国别/区域经济、比较制度、贸易投资、产业政策和地缘经济是核心
- 可用国别案例、跨国面板、贸易投资数据、政策文本和比较制度分析
- 需要避免新闻评论化，必须有理论框架或系统证据

## 结构与写作

- 引言说明国别/区域问题为何具有中国或全球意义
- 文献要覆盖区域研究与经济学/管理学理论
- 政策含义面向对外开放、区域合作或企业国际化

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 核对栏目、注释/参考文献、外文资料译名和投稿格式
- 确认国别数据、政策文件和翻译准确
- 检索“《亚太经济》 投稿须知 / 征稿简则 / 作者指南”，以最新页面为准；不要沿用二手代投网站格式
- 核对是否要求匿名正文、作者信息页、基金信息单列、数据/程序/补充材料、版权协议或不收费声明
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《亚太经济》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 方法、数据、图表和参考文献格式已按最新官方稿约复核
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 把时事评论当学术论文
- 只描述国别政策，缺比较框架
- 数据来源零散，无法复核

## 改投判断

日本专题转 contemporary-economy-of-japan；国际贸易计量转 journal-of-international-trade。

## 输出格式

```text
【匹配度】高 / 中 / 低（一句话理由）
【目标期刊】《亚太经济》
【选题标签】<最贴近的 2-3 个主题>
【方法证据】<当前方法是否够本刊标准>
【最大风险】<最可能导致退稿的一点>
【需核验官方要求】<投稿系统/匿名/摘要/图表/数据等>
【改投建议】<若不匹配，给出更合适期刊>
```
