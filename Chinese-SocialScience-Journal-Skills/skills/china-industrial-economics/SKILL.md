# 《中国工业经济》投稿（china-industrial-economics）

## 期刊定位

国内**产业经济 / 企业与创新实证**顶刊。以"规范实证"著称——它甚至发表过被高频引用的方法操作指南（中介效应、双重差分等的规范流程）。编委口味：**识别要干净、稳健性要做满、机制+异质性要齐、政策含义要落地。** 比《经济研究》更"实证工程化"，比《管理世界》更偏经济学计量。

## 触发时机

- 题目涉及产业组织、企业行为、创新研发、数字经济、区域经济、环境规制
- 用中国政策做准实验（试点、补贴、环保督察、宽带中国、低碳城市、自创区……）
- 担心稳健性章节不够厚、审稿人要求补一堆检验
- 机制/异质性是否到位拿不准

## 选题偏好

1. **政策评估类准实验**：某项产业/区域/环境政策对企业或地区结果的因果效应
2. **企业行为与绩效**：创新、出口、全要素生产率、数字化转型、ESG
3. **产业组织与竞争**：市场结构、进入退出、要素配置效率
4. **数字经济与新动能**：平台、数据要素、人工智能对生产率的影响

## 方法与识别（实证工程化）

- 主力是 **DID / 多期 DID / event-study**；必须做平行趋势 + 安慰剂 + 现代异质性稳健估计
- 机制检验：优先"分渠道回归 / 调节效应"，谨慎使用三步中介（编委对其有保留）
- 异质性：按企业所有制、规模、行业、地区、要素密集度等多维切分，并解释经济含义
- 稳健性"军备竞赛"：替换度量、剔除干扰政策、PSM-DID、改变窗口、安慰剂、排除竞争性解释——**做满是常态**
- 内生性：IV / Heckman / 双重差分组合，明确处理

## 结构与写作

- 引言：现实问题 + 政策背景 + 边际贡献 3–5 条
- 制度背景段要把政策细节讲准（文件、时点、试点名单）
- 主表后必有经济含义解读（量级、占均值比重），不只报显著性
- 政策含义**可操作**：谁、在哪个环节、做什么——这是本刊与《经济研究》的关键差异
- 篇幅充实，表多图精

## 官方核验清单

- 检索“《中国工业经济》 投稿须知 / 征稿简则 / 作者指南”，以官网、主办单位页面、采编系统或期刊官方 CNKI 页面为准
- 核对是否要求匿名正文、作者信息页、基金信息单列、摘要/关键词/JEL 或中图分类号、图表/参考文献格式
- 核对是否有数据、程序、补充材料、版权协议、不收费声明或特殊栏目要求
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] DID 平行趋势 + 安慰剂 + 现代异质性估计齐全
- [ ] 机制走"分渠道/调节"而非单纯三步中介
- [ ] 异质性多维且有经济解释
- [ ] 稳健性覆盖：换度量/换样本/PSM-DID/排他性解释
- [ ] 制度背景与政策时点准确
- [ ] 政策建议具体到主体与环节

## 高频拒稿雷区

- DID 不做平行趋势/安慰剂，或忽视交错处理偏误
- 机制只有三步中介且无理论渠道
- 异质性只切"东中西"一刀
- 稳健性单薄，审稿人一问就塌
- 政策建议是"加强、完善、推进"四件套

## 决策口诀

- "干净政策冲击 + 厚稳健性 + 落地政策" → 本刊正中
- "理论贡献厚于实证" → 转《经济研究》
- "偏管理理论/案例/问卷" → 转《管理世界》或《南开管理评论》
- "落点在金融机理" → 转《金融研究》

## 输出格式

```
【匹配度】高 / 中 / 低
【识别】DID/event-study 合规性：平行趋势 □ 安慰剂 □ 异质性稳健 □
【机制】分渠道/调节 √ | 仅三步中介（需改）
【异质性维度】<已切维度>（建议补 <维度>）
【稳健性清单】已做 <…> / 建议补 <…>
【政策可操作性】高 / 中 / 低
```
