# Source Basis and Official-Check Notes

This pack encodes journal-fit heuristics for the Chinese roadmap journals. It is
not a static substitute for live author instructions.

## How to use this resource

Every per-journal skill should force a live official-source check before giving
submission-ready advice. Use this priority order:

1. The journal's own official domain or official submission system.
2. A host institution / society page, including university, academy, association,
   institute, or official editorial-office pages.
3. A CNKI-hosted official journal page or official journal PDF.
4. Aggregator pages only as discovery aids, never as final authority.

When giving user-facing advice, name the live source checked and date-sensitive
items that still need re-checking. If official instructions conflict with a
skill, the official instructions win.

For per-journal starting queries, use `official-source-map.md` in this directory.

## Operational rule for every journal skill

1. Search the live official site or journal-hosted submission system for
   `期刊名 + 投稿须知` / `征稿简则` / `作者指南`.
2. Trust official journal domains, university/institute pages, society pages,
   CNKI-hosted official journal pages, or the journal's online submission system
   before third-party submission brokers.
3. Re-check anonymous-review rules, author-information handling, word limits,
   abstract/keywords/JEL or classification codes, reference style, figure/table
   rules, data/code requirements, copyright agreement, publication-charge policy,
   and anti-fraud / anti-agent notices.
4. Record uncertainty explicitly when the official page is unavailable, stale, or
   contradicted by the submission system.
