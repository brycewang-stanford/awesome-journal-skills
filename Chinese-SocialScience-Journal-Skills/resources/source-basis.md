# Source Basis and Official-Check Notes

This pack encodes journal-fit heuristics for the Chinese roadmap journals. It is not a static substitute for live author instructions.

Checked source types while building this expansion:

- Root repository roadmap: `README.zh-CN.md`, section `中文 —— 经管主流 100 本`.
- CSSCI 2023-2024 source journal directory mirrors and FMS 2025 management-science journal list pages.
- Representative official journal pages and author instructions, including CEQ, 世界经济, 国际金融研究, 金融研究, 管理科学学报, 经济科学, 会计研究, and 中国工业经济.

Operational rule for every journal skill:

1. Before final submission, search the live official site or journal-hosted submission system for `期刊名 + 投稿须知` / `征稿简则` / `作者指南`.
2. Trust official journal domains, university/institute pages, society pages, CNKI-hosted official journal pages, or the journal's online submission system before third-party submission brokers.
3. Re-check anonymous-review rules, author-information handling, word limits, abstract/keywords/JEL or classification codes, reference style, figure/table rules, data/code requirements, copyright agreement, and fee statement.
4. If official instructions conflict with a skill, the official instructions win.
