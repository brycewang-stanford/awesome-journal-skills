# 《中国农村经济》投稿检查清单

> 准确性提示：具体的字数、参考文献条数、图表数量等限制以《中国农村经济》官网"投稿须知"当年要求为准，下表为惯常量级。

## 投稿前格式检查

- [ ] 稿件为 DOC 或 DOCX 格式
- [ ] 稿件已完全匿名化处理（正文中无作者、单位、课题、调研团队等可识别信息）
- [ ] 文件属性中无作者信息
- [ ] 章节编号规范：一、（一）、1、依次嵌套
- [ ] 公式独立成行、居中、右侧带编号

## 作者信息检查

- [ ] 已明确标注通讯作者
- [ ] 作者署名单位、资助课题信息真实准确
- [ ] 致谢、基金信息单独提交，不出现在正文

## 摘要与关键词检查

- [ ] 中文摘要符合本刊规定字数，含数据 / 方法 / 量化发现
- [ ] 英文摘要与中文语义对齐
- [ ] 中文关键词 + 英文对应
- [ ] JEL 分类号已标注（如本刊要求）

## 正文结构检查

- [ ] 引言点明三农问题与现实背景
- [ ] 文献述评中外并重，理论与三农本土权威文献到位
- [ ] 本文边际贡献明确写出（3–5 条）
- [ ] 结论与政策含义严格演绎自前文证据
- [ ] 引言、文献、结论各部分不简单重复

## 内容质量检查

- [ ] 清晰说明与已有文献（特别是本刊及《中国农村观察》已发表文章）的学术贡献区别
- [ ] 识别策略可靠，处理了农户自选择内生性
- [ ] 机制分析落到农户 / 村庄微观层面，非纯宏观叙事
- [ ] 异质性维度贴合三农场景（非仅"东中西"）
- [ ] 政策含义落到具体三农对象，无"加强完善推进"四件套
- [ ] 无空洞的"有意义""有价值"等表述

## 图表与公式检查

- [ ] 主表列数克制（≤ 6 列），三线表格式、无竖线
- [ ] 标准误格式统一，注明聚类层次
- [ ] 变量定义表给出计算口径、单位、数据来源（点名调查 / 数据库）
- [ ] 图表有清晰标题和来源说明
- [ ] 图表编号连续且与正文引用对应

## 参考文献检查

- [ ] 采用本刊统一格式（投稿须知核对）
- [ ] 中文文献、外文文献分别按规定排序
- [ ] 文中引用与文末列表一一对应
- [ ] 三农理论与本土权威文献齐全
- [ ] 对话了本刊及姊妹刊近 3 年同主题文章
- [ ] 引用内容真实、完整

## 实证研究专项检查

- [ ] 数据来源权威（CFPS / CHFS / CLDS / 农村固定观察点 / 农业农村部统计等）
- [ ] 自行调研的，交代抽样框、年份、样本量
- [ ] 样本选择标准清晰
- [ ] 变量定义清晰明确
- [ ] 识别策略说明清楚（DID / IV / RDD / PSM-DID）
- [ ] 农户自选择 / 内生性问题有处理
- [ ] 稳健性、机制、异质性检验完备

## 投稿系统检查

- [ ] 在本刊在线投审稿系统注册并完善信息
- [ ] 双向匿名评审要求已满足（稿件与系统信息均匿名）
- [ ] 选择正确的栏目 / 研究类型
- [ ] 稿件是最新版本
- [ ] 仔细核对系统中显示的相关信息
