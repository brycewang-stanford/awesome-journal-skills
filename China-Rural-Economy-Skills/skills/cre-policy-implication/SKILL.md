# 政策含义（cre-policy-implication）

## 触发时机

- 文末政策建议只有"建议政府重视乡村振兴"
- 政策建议是"加强、完善、推进、深化"四件套套话
- 审稿人指出"政策含义太薄 / 与本文证据脱节"

## 本刊政策含义的定位

《中国农村经济》的政策含义偏**"意义层"**——是研究结论对乡村振兴 / 农村政策的制度含义，而**不是**给某部委的执行备忘录。但"意义层"不等于空泛：必须**落到具体的三农场景与对象**（哪类农户、哪个环节、哪项制度），与本文实证证据严格挂钩。

| 维度 | 合格（本刊偏好）                       | 不合格                       |
|------|--------------------------------|----------------------------|
| 层面 | 制度 / 原理层，但点明三农对象           | 笼统"政府应重视"             |
| 落点 | 哪类农户 / 哪个环节 / 哪项政策          | 不指对象的"加强完善推进"      |
| 依据 | 与本文机制 / 异质性结论呼应             | 与本文证据无关的泛泛建议       |
| 收尾 | 与理论贡献 + 乡村振兴议题呼应           | 复述政府工作报告              |

## 结构

### 三段式（推荐）

```
本文的研究具有以下政策含义。

第一，[原理层 + 三农落点]——本文揭示了[机制]，提示在[土地 / 补贴 / 金融 / 组织等]制度设计中应重视……。
结合本文异质性发现，对[某类农户 / 某类地区]而言尤其如此。

第二，[宏观 / 战略层]——本文发现……，这意味着乡村振兴 / 粮食安全 / 共同富裕在……方面仍有改进空间。

第三，[微观主体层]（如适用）——对农户 / 合作社 / 家庭农场而言，本文提示……。
```

### 单段式（如果文章偏理论）

```
本文为理解中国农村[领域]的[现象 / 制度]提供了新的微观证据。
本文发现……，这一结果意味着[制度意义]，对[乡村振兴 / 农业政策 / 相关理论]具有……启示。
```

## 必查清单

- [ ] 政策含义与本文实证证据 / 机制 / 异质性紧密挂钩
- [ ] 不出现"加强 / 完善 / 推进 / 深化"四件套撑段落
- [ ] 含义落到**具体的三农对象**（哪类农户 / 哪个环节 / 哪项制度），不是笼统"政府"
- [ ] 与引言中的"研究意义"段相呼应
- [ ] 呼应乡村振兴 / 粮食安全 / 共同富裕等战略议题，但不止于复述口号

## 反模式

- 政策建议用"加强、完善、推进、深化"四个动词撑满段落
- 把政策含义写成"政府工作报告"或"乡村振兴文件"的延伸
- 含义与本文研究无关（研究的是土地流转，建议却谈农村教育）
- 文末才提到农村 / 乡村振兴 —— 三农问题意识应贯穿全文
- "本文是首次……" —— 这是贡献，不是政策含义

## 输出格式

```
【段数】X
【层级覆盖】原理 / 宏观战略 / 微观主体
【三农落点】明确（哪类农户 / 环节 / 制度）/ 笼统
【与本文证据挂钩度】高 / 中 / 低
【四件套套话命中】X 处
【下一步】cre-abstract
```
