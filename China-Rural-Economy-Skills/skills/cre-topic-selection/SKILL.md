# 中国农村经济选题（cre-topic-selection）

## 触发时机

- 用户说"我想冲《中国农村经济》，但选题不确定是否够'三农'"
- 边际贡献写不出来 / 写了一段但和审稿人能 buy 的角度不在一个频道
- 不确定该投《中国农村经济》、《中国农村观察》还是综合性经济学期刊

## 三个硬约束

《中国农村经济》对选题的硬约束：

1. **三农场景**：选题必须真正落在农业经济、农村发展或农民 / 农户行为上，第一段就要说明"为什么这是一个农村问题、为什么现在做"
2. **理论贡献 + 政策现实并重**：仅有"现象 + 描述统计 + 政策"不够，要能挂上一支理论（农户行为、土地制度、农业产业组织、发展经济学、农村金融等），同时回应乡村振兴 / 粮食安全等政策现实
3. **边际贡献可枚举**：能拆成 3–5 条独立的、互不重叠的贡献

## 选题范围自检（落到三农）

本刊关注的主流议题（用来判断"是否对口"）：

- 农业经济与农业生产经营（农业经营主体、规模经营、农业全要素生产率）
- 土地制度（确权、流转、"三权分置"、宅基地）
- 粮食安全与农业政策（补贴、价格、农业支持保护）
- 城乡融合与城镇化（要素双向流动、户籍）
- 乡村振兴战略（产业、人才、文化、生态、组织五个振兴）
- 农户行为与农业经营主体（合作社、家庭农场、龙头企业）
- 农村金融（信贷、保险、数字普惠金融）
- 扶贫 / 巩固脱贫成果与共同富裕
- 农村劳动力与人口（外出务工、留守、老龄化、回流）

如果选题在上述清单里找不到落点，**多半不对口本刊**。

## 理论贡献自检

写完一句话摘要后，逐条自检：

- [ ] 本文挂在哪一支理论上？（农户行为 / 土地制度 / 契约与组织 / 发展经济学 / 农村金融 / 劳动经济）
- [ ] 该理论分支的代表性文献是哪 2–3 篇（含三农领域中文经典与国际理论）？
- [ ] 本文相对于该理论的边际差异是什么？（修正 / 扩展 / 引入新机制 / 提供首个农村微观证据）
- [ ] 中国农村情境是否提供了海外文献没有的识别机会（如某项农村政策的准实验冲击）？

少于 3 个 yes → 选题深度不够，回到三农理论文献再读两轮。

## 边际贡献四句式

```
本文的边际贡献主要体现在以下四个方面：
第一，[理论维度]——本文将……纳入[农户行为 / 土地制度 / 农村金融等理论框架]，区别于[已有研究]……。
第二，[识别策略维度]——本文利用[农村政策 / 制度冲击]构造……，缓解了农户自选择（如入社 / 外出务工）带来的内生性问题。
第三，[机制维度]——本文揭示了农户层面的……机制，回答了已有文献"知其然不知其所以然"的问题。
第四，[政策意义维度]——本文为理解[乡村振兴 / 粮食安全 / 共同富裕等议题]提供了……微观证据。
```

四条都齐才算合格。少于四条 → 选题深度不够。

## 期刊匹配判断

|              | 中国农村经济 | 中国农村观察 | 综合经济学期刊 |
|----------|--------|--------|------------|
| 学科        | 农业 / 农村经济（实证为主） | 三农（含质性 / 案例 / 政策观察） | 一般经济学 |
| 三农场景    | **必备**  | **必备**  | 不要求       |
| 因果识别    | 高门槛   | 较包容   | 高门槛       |
| 案例 / 质性 | 较少     | 接受     | 较少         |
| 政策含义    | 意义层（落到三农） | 意义层 / 观察 | 意义层    |

如果稿子脱离三农场景，**不要投《中国农村经济》**；偏质性 / 案例可考虑《中国农村观察》。
