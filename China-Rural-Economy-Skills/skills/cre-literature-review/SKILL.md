# 中国农村经济文献综述（cre-literature-review）

## 触发时机

- 综述缺三农领域中文经典，或全部是英文文献
- 综述缺**理论文献**（农户行为、土地制度、契约组织等理论根基）
- 综述写成了"罗列式"而非"对话式"

## 结构规范

《中国农村经济》偏好**三层结构**的综述：

1. **第一层：理论文献**（中外并重）——建立问题的理论坐标（如农户行为理论、诱致性制度变迁、契约理论、发展经济学）
2. **第二层：三农领域实证文献**（中文为主、英文为辅）——梳理已有的农村微观实证发现
3. **第三层：本文位置**——明确指出"本文与现有研究的差异"，尤其是相对于本刊与姊妹刊《中国农村观察》近年同主题文章的差异

### 与综合经济学期刊综述的差异

《中国农村经济》尤其看重**三农领域文献是否到位**：既要有国际理论与方法文献，也要充分对话国内农经学界的权威研究。一篇只引国际顶刊、不引国内农经经典的稿件，会被审稿人认为"未扎根本土文献"。

## 中外文献配比

经验法则（具体不必拘泥，导向是"中外并重 + 理论扎实"）：

- 总引文 40–80 条（投稿前请到官网"投稿须知"核对当年参考文献条数上限）
- 中文文献占比 **40%–60%**（三农领域本土实证不能缺位）
- **理论文献占比 ≥ 20%**（理论推导 / 综述，不是实证应用）
- 五年内文献占比 **≥ 40%**
- 既覆盖国际顶刊（AJAE / AER / JDE / Food Policy 等），也覆盖国内权威（《经济研究》《管理世界》《中国农村经济》《中国农村观察》《农业经济问题》等）

## 必引避坑清单

- [ ] 是否引用了所属理论分支的**经典理论文献**（农户行为 / 土地制度 / 契约组织 / 发展经济学的原始 paper）？
- [ ] 是否引用了**三农领域的中文权威研究**（不是只引国际文献）？
- [ ] 是否引用了近 3 年本刊或《中国农村观察》的同主题论文？
- [ ] 引用是否覆盖三类：理论、方法、本土三农实证？
- [ ] 是否避免了"自引扎堆"？

## 综述句式模板

```
[作者 (年份)] 利用[农户 / 村级数据 + 方法]研究了[三农问题]，发现……。
然而，该研究[局限]，未考虑[本文要补的空白，如农户自选择 / 区域异质 / 机制]。
与之相比，本文……。
```

每段综述至少出现一次"与之相比 / 区别于 / 本文延伸了 / 本文补充了"。

## 对话式 vs 罗列式

《中国农村经济》偏好**对话式**综述（研究之间相互回应），而非**罗列式**（按作者顺序点名）。

### 错误（罗列式）

> 张三（2020）研究了土地流转，李四（2021）研究了农户增收，王五（2022）研究了合作社。

### 正确（对话式 — 按主题分组、表明分歧、给出本文位置）

> 土地流转能否提升农户收入，已有研究存在分歧（张三，2020；李四，2021）：一类研究认为流转通过规模经营提高生产率，另一类则强调流转中的契约不完全限制了增收效果。两类判断的差异部分源于对农户自选择的处理不同。本文沿这一思路，使用[数据]并构造[识别策略]，进一步考察了……。

判断标准：把作者名都隐去后，综述还能读出"研究 A 怎么说、研究 B 怎么反驳、本文怎么定位"，就是对话式。

## 与本文贡献阐述衔接

综述末尾必须导出本文的差异化定位，模板：

> 本文与现有文献的重要区别体现在以下几个方面：
> 第一，[识别策略上的差异——如何处理农户自选择 / 政策内生]；
> 第二，[数据或样本上的差异——更新 / 更细的农户或村级数据]；
> 第三，[机制或异质性上的差异]。
> 与本刊（或《中国农村观察》）近期发表的 XXX（作者，202X）一文相比，本文 ……

**与本刊及姊妹刊已发表文章对比**是审稿人的关注点——综述若不提近 3 年两刊同主题文章，会被视为"未充分对话"。

## 反模式

- 把综述写成"三农政策背景介绍"而非学术对话
- 综述完全围绕国际文献，无国内三农权威研究
- 中文文献只引一两条凑数
- 综述里出现一长串并列引用 —— 拆开按贡献分述
- 自引扎堆

## 输出格式

```
【结构】三层 / 两层 / 一层
【中外配比】中文 X% / 英文 Y%
【理论文献占比】X%
【三农本土权威文献】到位 / 缺位
【五年内文献比例】X%
【缺漏清单】[...]
【下一步】cre-identification
```
