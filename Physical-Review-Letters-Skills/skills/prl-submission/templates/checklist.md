# Physical Review Letters — Pre-Submission Checklist

> Verify all volatile specifics (length limit, length formula, classification scheme,
> portal URL, required statements) on the official APS / PRL author page before submitting.

## 1. Scope and venue

- [ ] The result clears the importance + broad-interest gate (see `prl-scope-fit`)
- [ ] You have honestly considered Phys. Rev. A–E / PR Research as alternatives
- [ ] The single central claim is stated in one sentence

## 2. Length (the deductible budget)

- [ ] Manuscript passes the current APS automated length check
- [ ] Body + figures + equations + references counted together against the limit
- [ ] No double-column figure unless the data require it
- [ ] Only claim-defining equations are displayed inline
- [ ] Reference list pruned to importance + trust essentials
- [ ] Abstract within the current length limit

## 3. Format

- [ ] Prepared in REVTeX (APS LaTeX class) or an accepted format
- [ ] Submitted as a **Letter** (not a regular article)
- [ ] SI units; consistent significant figures
- [ ] Every symbol defined on first use; notation consistent across Letter / figures / SM
- [ ] Acronyms spelled out on first use; abstract is jargon-light

## 4. Figures

- [ ] A few high-impact figures only; each is essential
- [ ] Fig. 1 conveys the central result at a glance
- [ ] Legible at single-column print size (fonts ~body size)
- [ ] Color-blind-safe and grayscale-distinguishable; not color-dependent
- [ ] Vector source for line art; high-resolution raster for images
- [ ] Captions define all symbols and are as short as clarity allows

## 5. Supplemental Material

- [ ] The Letter stands alone without the SM
- [ ] No central-claim evidence lives only in the SM
- [ ] Every "see Supplemental Material" pointer resolves
- [ ] SM figures/tables use the S-prefix (Fig. S1, Table S1) and are cited from the body
- [ ] SM provided as a separate file, not appended to the Letter PDF

## 6. Files to upload

- [ ] Main Letter source (LaTeX/REVTeX) + compiled PDF
- [ ] All figure files at production resolution
- [ ] Supplemental Material file (separate, labeled)
- [ ] Cover letter (justifies importance + broad interest)
- [ ] Bibliography / .bib as required

## 7. Classification and metadata

- [ ] Subject area / editor track selected accurately
- [ ] Current APS subject classification applied
- [ ] Title / abstract / author list consistent with the manuscript
- [ ] Funding / acknowledgments per APS instructions
- [ ] Data-availability and any required statements present

## 8. Authors and integrity

- [ ] All authors meet authorship criteria and approved submission
- [ ] Corresponding author and contact details correct
- [ ] ORCID and affiliations complete
- [ ] Originality / not-under-consideration-elsewhere confirmed
- [ ] Prior arXiv / conference version disclosed
- [ ] Competing-interest statement if required

## 9. Referees

- [ ] Suggested referees can judge correctness AND breadth
- [ ] At least one suggested referee from the claimed neighboring field
- [ ] No conflicts of interest among suggestions
- [ ] Opposed referees (short list) with brief, factual, professional rationale

## 10. Self-consistency

- [ ] Figure / equation / reference numbering continuous and matching in-text callouts
- [ ] Cover-letter referee section matches the submission form
- [ ] Cross-references intact after any last-minute trimming
