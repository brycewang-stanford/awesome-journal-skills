# JPE Pre-Submission Checklist

Verify the *current* JPE author guidelines, submission fee, and data/code policy on the
journal's official page before relying on any limit below. This list captures the durable
structure of a Journal of Political Economy submission via Editorial Express.

## 1. Format
- [ ] Manuscript is a single PDF, double-spaced
- [ ] Section order is logical: introduction → model/theory → data & setting → empirical strategy → results → mechanisms/robustness → (counterfactuals) → conclusion
- [ ] Equations numbered; referenced in text as "equation (n)"
- [ ] Tables are self-contained (sample, period, unit, DV definition, specification, clustering level in notes)
- [ ] Figures legible in grayscale, axes labeled with units, confidence bands shown
- [ ] Online appendix prepared as a separate file (proofs, full robustness, descriptive tables)

## 2. References (house style)
- [ ] Author-date citations throughout — NOT numbered/bracket citations
- [ ] Reference list complete, alphabetical, consistent Chicago/UChicago Press style
- [ ] Every in-text citation appears in the list; every list entry is cited at least once
- [ ] Canonical theory the result engages is cited (common referee snag)

## 3. Economic argument (substance gate)
- [ ] An explicit economic mechanism or model is stated, not just a net effect
- [ ] The estimated object has a clear economic interpretation
- [ ] General-equilibrium / selection / incentive considerations addressed or bounded
- [ ] The leading rival mechanism is named and tested against
- [ ] The portable economic lesson is stated, not assumed

## 4. Anonymity (if review is double-blind — verify current policy)
- [ ] Author identities removed from manuscript body
- [ ] PDF metadata scrubbed of author names
- [ ] Self-citations phrased neutrally ("Smith (2020) shows...")
- [ ] Acknowledgments, funding, and institutional info on a separate title page only

## 5. Fee & policy
- [ ] Submission fee amount confirmed on the official page (check waivers/discounts)
- [ ] Payment method ready (paper typically not forwarded until fee clears)
- [ ] Data and code availability policy reviewed (AEA Data Editor standard)
- [ ] Replication package staged for the accepted-paper stage
- [ ] Paper not under review elsewhere; no overlapping submission

## 6. Exhibits & supplementary
- [ ] Main-text exhibits report economic magnitudes (mean-relative / elasticity / welfare), not stars alone
- [ ] Number of main-text exhibits is disciplined; heavy material moved to the online appendix
- [ ] Online appendix referenced from the main text and uploaded
- [ ] Restricted-data documentation prepared if applicable

## 7. Portal metadata (Editorial Express)
- [ ] Title and abstract entered
- [ ] JEL classification codes entered
- [ ] Keywords entered
- [ ] All co-authors and affiliations correct; corresponding author set
- [ ] Suggested referees (and any opposed referees) prepared with brief justification

## 8. Final pass
- [ ] Cover letter drafted (one page: question, contribution, fit with JPE)
- [ ] Manuscript number / acknowledgment email expected after submission
- [ ] Replication package re-run on a clean machine (for the accepted stage)
