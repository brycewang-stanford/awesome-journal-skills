# SMJ Manuscript Skeleton

> A structural template for a Strategic Management Journal empirical paper. Adapt the arc for qualitative theory-building or formal/analytical work, but foreground the strategy contribution early in all cases. Confirm length, abstract limit, and reference style against the **current SMJ Author Guidelines** (Wiley / Strategic Management Society) — these change periodically.

## Title

> [Concise, informative title naming the strategic phenomenon and outcome]

## Abstract

> One paragraph (verify the current word limit). State: (1) the strategy question, (2) the theoretical lens / argument, (3) the data and identification approach, (4) the key finding, (5) the contribution to strategy theory. No citations.

**Keywords:** [e.g., competitive advantage; dynamic capabilities; endogeneity; alliances] (follow current guideline count)

## 1. Introduction

- Motivate the strategy question (why it matters for firm performance / competitive advantage).
- Name the scholarly conversation and the unresolved gap (mechanism / tension / contingency).
- State the contribution to strategy theory in 1–2 sentences ("we show that…").
- Preview the design, the identification approach, and the headline finding.
- (Reader should know the contribution by the end of page two.)

## 2. Theory and Hypotheses

### 2.1 Theoretical background
> The lens you build on (RBV / dynamic capabilities / TCE / agency / competitive dynamics / BTOF / real options / institutional) and where you extend it.

### 2.2 Hypothesis development
> Baseline relationship with its mechanism, then mediation/moderation. Each hypothesis preceded by an explicit causal logic; acknowledge and address a competing prediction.

- **H1 (baseline):** … *(mechanism: …)*
- **H2 (mechanism/mediation):** …
- **H3 (moderation):** … *(why the boundary condition shifts the mechanism)*

## 3. Methods

### 3.1 Sample and data
> Population, sampling frame, time window, unit of analysis (firm-year / BU-year / alliance / deal). All filters and their justifications (survivorship, missingness).

### 3.2 Measures
| Role               | Variable | Operationalization | Source |
|--------------------|----------|--------------------|--------|
| Dependent (performance) | | | |
| Focal independent (strategic choice) | | | |
| Moderator(s)       | | | |
| Mechanism / mediator | | | |
| Controls           | | | |

### 3.3 Identification strategy
> Name the identifying threat (selection into the strategic choice / reverse causality / unobserved heterogeneity) and the design that defeats it (FE + IV / DID / matching / Heckman). State the estimating equation and the clustering level.

## 4. Results

### 4.1 Descriptive statistics and correlations
> One table; note multicollinearity.

### 4.2 Main results
> Nested/hierarchical columns: controls → focal X → moderators/mechanism.

### 4.3 Identification and endogeneity
> The evidence that addresses the threat: parallel-trends/event study (DID), first-stage and exclusion argument (IV), balance and common support (matching), exclusion restriction (Heckman). Placebo/falsification tests.

### 4.4 Mechanism test
> Direct evidence on the theorized mechanism, beyond a single mediation regression.

### 4.5 Robustness
> Alternative DV operationalizations, samples, estimators, and clustering.

## 5. Discussion

### 5.1 Contribution to strategy theory
> What strategy scholars should now believe differently; tie to the named conversation.

### 5.2 Managerial implications
> Proportionate to the theoretical contribution; do not let it dominate.

### 5.3 Limitations and future research
> Honest residual threats (e.g., remaining selection on unobservables) and a forward agenda.

## References

> Current SMJ/Wiley reference style. Every in-text citation in the list and vice versa; cite recent relevant SMJ work.

## Online Appendix / Supplementary Materials

> Secondary tables, additional robustness, variable construction details, data-availability information.
