# SMJ Pre-Submission Checklist

> Verify all volatile specifics (word limit, fee/open-access options, required statements, reference style) on the **current SMJ Author Guidelines** before submitting. SMJ is published by Wiley for the Strategic Management Society and uses ScholarOne Manuscripts. This checklist covers the durable items.

## 1. Scope & contribution

- [ ] The focal question is a strategy question (firm performance / competitive advantage)
- [ ] The contribution to strategy theory is stated explicitly in the introduction
- [ ] The contribution is specific and falsifiable, not "contributes to the literature"
- [ ] The paper would update what strategy scholars believe (not generic "management")

## 2. Theory & positioning

- [ ] Every hypothesis is preceded by an explicit causal mechanism
- [ ] The paper enters a named scholarly conversation and cites the closest recent SMJ work
- [ ] The 2–3 closest papers are differentiated explicitly
- [ ] Moderators are theorized, not bolted on

## 3. Methods & identification (most scrutinized)

- [ ] Sample, time window, and unit of analysis are explicit and matched to theory
- [ ] All sample filters reported and justified (survivorship / missingness)
- [ ] Performance DV justified; alternatives identified for robustness
- [ ] Strategic-choice measure validated and sourced
- [ ] The identifying threat is named (selection / reverse causality / unobserved heterogeneity)
- [ ] A design defeats that threat (IV, DID/natural experiment, matching, Heckman, FE)
- [ ] DID: parallel-trends evidence shown; staggered-timing bias addressed; placebos run
- [ ] IV: first-stage strength reported; exclusion argued in prose; reduced form shown
- [ ] Matching: balance + common support shown; unobservable selection acknowledged
- [ ] Mechanism tested, not just asserted
- [ ] Standard errors clustered at the assignment level; robustness battery reported

## 4. Exhibits

- [ ] Each main-text table is self-contained (title + complete notes)
- [ ] Standard errors shown; clustering level stated in notes
- [ ] Interactions visualized with a marginal-effects plot
- [ ] Event-study figure present if using DID
- [ ] N and variable naming consistent across tables; secondary tables in supplement

## 5. Writing & structure

- [ ] Structure: intro → theory → methods → results → discussion → references
- [ ] Contribution visible by end of the introduction's second page
- [ ] Discussion interprets (implications + limits + agenda), not re-lists results
- [ ] Constructs named consistently; hedging proportionate to evidence
- [ ] Strategy terminology used precisely

## 6. Anonymization (double-blind)

- [ ] Author names, affiliations, acknowledgments removed from the manuscript file
- [ ] Self-citations phrased neutrally (not "our earlier paper")
- [ ] File metadata/properties scrubbed; file names contain no identity
- [ ] Funding/acknowledgments only where the system requests them

## 7. Files & metadata (ScholarOne)

- [ ] Anonymized main manuscript
- [ ] Separate title page with author details (if required)
- [ ] Cover letter (states the strategy contribution; confirms originality and exclusivity)
- [ ] Online appendix / supplementary materials (if any)
- [ ] Required statements present (data availability, conflict of interest, ethics) — confirm current requirements
- [ ] Correct article type selected (regular vs. special issue)
- [ ] Suggested / opposed reviewers entered if invited

## 8. References

- [ ] Reference style matches current SMJ/Wiley guidelines (verified, not assumed)
- [ ] Every in-text citation appears in the list and vice versa
- [ ] Recent relevant SMJ work is cited; nothing obvious is missing
- [ ] Citation-manager default style replaced with the journal style
