# NEJM Submission Preflight Checklist

> Copy this file, tick every box, and resolve all blocking items before upload.
> Targets reflect typical NEJM / ICMJE guidance — confirm exact caps in the current
> NEJM Author Center and ICMJE Recommendations.

## 1. Significance & venue
- [ ] Practice-changing clinical impact (nejm-fit rung ≥ 4; clears all three gates)
- [ ] Patient-important primary outcome (not surrogate-only)
- [ ] Article type: ☐ Original Article  ☐ Brief Report  — within length/reference caps

## 2. Registration, protocol & SAP
- [ ] Trial prospectively registered BEFORE enrollment: registry ______ number ______
- [ ] Registered primary outcome matches the reported primary outcome
- [ ] Protocol finalized & submission-ready
- [ ] Statistical analysis plan (SAP) pre-specified, dated, submission-ready
- [ ] Any protocol/SAP deviation disclosed and explained

## 3. Reporting guidelines (EQUATOR)
- [ ] Guideline: ☐ CONSORT (☐ extension) ☐ STROBE ☐ PRISMA ☐ SPIRIT ☐ CARE ☐ STARD
- [ ] Completed checklist with page/line references
- [ ] CONSORT participant flow diagram (RCT) / PRISMA selection diagram (SR-MA)
- [ ] Flow-diagram numbers reconcile with Table 1 and analysis populations

## 4. Structured abstract
- [ ] Four sections: Background / Methods / Results / Conclusions
- [ ] ≤ 250 words: ______ words
- [ ] Primary outcome with effect estimate + 95% CI
- [ ] Intention-to-treat + per-group n stated
- [ ] Trial registration number (NCT…) present
- [ ] Funding source stated

## 5. Statistics
- [ ] Effect estimates + 95% CI (not P alone); exact P values
- [ ] ITT primary; per-protocol as sensitivity (both for non-inferiority)
- [ ] Multiplicity controlled; exploratory endpoints labeled
- [ ] Subgroups pre-specified; interaction tests reported (forest plot)
- [ ] Absolute risk difference + NNT alongside relative measures
- [ ] Missing-data handling stated; non-inferiority margin justified (if applicable)
- [ ] Causal language appropriate to design (cautious for observational)

## 6. Display items
- [ ] Table 1 by group; NO baseline P values (standardized differences if used)
- [ ] Kaplan–Meier with numbers-at-risk row; HR + 95% CI shown
- [ ] Forest plots for subgroups / meta-analysis (heterogeneity reported)
- [ ] Figures de-identified (no PHI); CIs shown; colorblind-safe; standalone legends

## 7. Ethics & integrity
- [ ] IRB / ethics committee approval stated
- [ ] Informed consent described (or waiver justified)
- [ ] Declaration of Helsinki / GCP stated
- [ ] ICMJE disclosure forms collected for all authors
- [ ] Competing-interests statement drafted
- [ ] ICMJE authorship criteria met by each author; contributors acknowledged
- [ ] Medical writers disclosed with funding
- [ ] Role-of-the-funding-source statement; data access/vouching (sponsored trials)
- [ ] Data-sharing statement (what / when / to whom / how)

## 8. References
- [ ] Vancouver / ICMJE numbered style, ordered by appearance
- [ ] First six authors then et al. (confirm cutoff)
- [ ] NLM / Index Medicus journal abbreviations
- [ ] Reference count within cap: ______
- [ ] All in-text superscript numbers resolve; no gaps/duplicates

## 9. Required files & metadata
- [ ] Title page (title, authors, affiliations, corresponding author, word counts)
- [ ] Structured abstract + IMRAD main text + references
- [ ] Tables (incl. Table 1) + figures with legends + CONSORT flow diagram
- [ ] Supplementary appendix: protocol + SAP + supp tables/figures
- [ ] Completed reporting checklist (CONSORT/STROBE/PRISMA)
- [ ] Cover letter (clinical importance + what's practice-changing)
- [ ] ORCIDs; CRediT / author contributions
- [ ] Disclosure forms; funding statement; data-sharing statement

## 10. Final integrity sweep
- [ ] No over-claiming beyond the evidence (abstract Conclusions + Discussion)
- [ ] Registration number, denominators, primary-outcome numbers consistent everywhere
- [ ] Blinding convention matches the journal (NEJM typically single-blind)
