# AMJ Pre-Submission Checklist

> Verify all current limits, required files, fees, and policies on the official AMJ submission page and the AOM Style Guide before submitting. The items below capture durable expectations.

## 1. Anonymization (double-blind)

- [ ] Main manuscript file contains no author names or affiliations
- [ ] Self-citations are neutral ("prior research (Author, 2020)"), never "our prior study"
- [ ] Acknowledgments, funding, and bios are on the separate title page only
- [ ] File metadata/properties scrubbed of author identity
- [ ] File names contain no author/institution identifiers

## 2. Format (per current AOM Style Guide)

- [ ] Separate title page: title, authors, affiliations, contact, acknowledgments, funding
- [ ] Abstract within the current AMJ word limit
- [ ] Keywords supplied as required
- [ ] AOM/APA-style in-text citations and reference list, alphabetized
- [ ] AOM heading hierarchy; double-spaced; standard font and margins
- [ ] Manuscript length within the current AMJ expectation

## 3. Files for ScholarOne

- [ ] Anonymized main document (manuscript, tables, figures, as the portal specifies)
- [ ] Separate identified title page
- [ ] Cover letter (fit, contribution, not under review elsewhere)
- [ ] Supplementary / online appendix file(s) if applicable
- [ ] High-resolution figure files if requested separately

## 4. Declarations & ethics

- [ ] IRB/human-subjects approval stated where applicable (off the anonymized text)
- [ ] Data-availability / transparency statement per current policy
- [ ] No concurrent submission; prior working-paper/conference versions disclosed in cover letter
- [ ] Conflicts of interest and funding disclosed on the title page
- [ ] All authors agree to authorship and contribution roles

## 5. Measurement & analysis (empirical core)

- [ ] Reliabilities (alpha/CR), CFA fit, AVE, and discriminant validity reported
- [ ] Common-method bias addressed by design and statistics
- [ ] Estimator matches design (SEM / HLM / panel-FE / experiment / IV-DiD); SEs clustered correctly
- [ ] Endogeneity strategy executed for archival causal claims
- [ ] Mediation via bootstrap CIs; moderation via simple slopes + incremental variance
- [ ] Robustness checks and alternative-explanation tests included

## 6. Tables & figures

- [ ] Table 1 = means, SDs, correlations with reliabilities on the diagonal
- [ ] Result tables use nested models with ΔR²/fit reported
- [ ] Significance markers and SE type defined in every note; one-/two-tailed stated
- [ ] Every significant interaction has a simple-slopes plot
- [ ] Theoretical-model figure maps one-to-one to the hypotheses
- [ ] Each exhibit is self-contained; counts/format within current AMJ requirements

## 7. References & internal consistency

- [ ] Every in-text citation is in the reference list and vice versa
- [ ] Canonical theory works for the focal domain are cited
- [ ] Hypothesis labels, construct names, and table/figure numbers consistent throughout
- [ ] Intro's promised contribution matches the discussion's delivered contribution

## 8. Portal (ScholarOne)

- [ ] Author account and ORCID current
- [ ] Correct article type selected
- [ ] Best-fitting division/keywords selected for routing
- [ ] Suggested (non-conflicted) reviewers entered if requested
- [ ] Built PDF previewed and verified before final submission
