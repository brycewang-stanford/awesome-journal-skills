# PNAS Submission Preflight Checklist

> Copy this file, tick every box, and resolve all blocking items before upload.
> Targets reflect typical PNAS guidelines — confirm exact caps in the current PNAS Author Guidelines.

## 1. Significance & venue
- [ ] Broad significance / high quality established (pnas-fit; cross-division reader passes)
- [ ] PNAS is the right venue (not better at a field journal or Science/Nature)
- [ ] Title declarative, specific, parseable by a non-specialist

## 2. Submission track
- [ ] Track: ☐ Direct Submission  ☐ Contributed (NAS member)
- [ ] If Contributed: member within annual quota (≈2/year — confirm): ______
- [ ] If Contributed: ≥2 independent reviewers secured; reviews + author responses ready
- [ ] If Direct: suggested editors / reviewers prepared for the cover letter

## 3. Significance Statement & abstract
- [ ] Significance Statement ≤ 120 words: ______ words
- [ ] Significance Statement written for a non-specialist; distinct from abstract
- [ ] Abstract ≤ ~250 words: ______ words
- [ ] Abstract: single paragraph, no subheadings / no citations
- [ ] ≥ 1 quantified headline result in abstract

## 4. Structure, classification & length
- [ ] Order: Title / authors / Significance / Abstract / Intro / Results / Discussion / Materials and Methods (in-text) / refs
- [ ] Materials and Methods present in the MAIN TEXT (extended detail → SI Appendix)
- [ ] Classification — major division: ☐ Biological ☐ Physical ☐ Social Sciences
- [ ] Classification — minor subject area: ______
- [ ] Keywords (≤ ~5): ______
- [ ] Package within length/page budget: ______ (confirm cap)

## 5. Figures & display items
- [ ] Figures + tables within page/item allowance: ______
- [ ] Sized to 9 / 11.4 / 18 cm; min font ≥ ~6–8 pt
- [ ] Data shown (points + n); error bars defined (SD/SEM/CI)
- [ ] Scale bars on images; uncropped key blots + source data retained
- [ ] Colorblind-safe palette; meaning not by color alone; RGB/CMYK handled
- [ ] Each legend: title sentence + per-panel + statistics

## 6. Statistics & reproducibility
- [ ] Per claim: effect size + 95% CI + n + test + assumptions
- [ ] Biological vs technical replication distinguished
- [ ] Sample-size rationale / power stated
- [ ] Randomization & blinding reported or justified
- [ ] Multiple-comparison correction where needed
- [ ] Analysis code public + archived with DOI

## 7. Data, materials, ethics
- [ ] Data deposited; accession numbers / DOIs in hand: ______
- [ ] Data Availability Statement compliant (no "on request" for primary data)
- [ ] Materials/reagent sharing plan (Addgene / MTA)
- [ ] Ethics: IRB ☐  IACUC ☐  field/biodiversity permits ☐  N/A ☐

## 8. References
- [ ] PNAS numbered style, ordered by appearance
- [ ] Single numbered list
- [ ] Full author lists; journal abbreviations
- [ ] All in-text reference numbers resolve; no gaps/duplicates

## 9. Required files & metadata
- [ ] Main text (title, Significance Statement, abstract, body, Materials and Methods, references)
- [ ] Figures at required resolution + figure legends; tables
- [ ] SI Appendix (extended methods, supporting figs/tables, supplementary text); Datasets if any
- [ ] Cover letter (significance + track + suggested editor/reviewers if Direct)
- [ ] Authors, affiliations, ORCIDs, corresponding author
- [ ] Author contributions
- [ ] Competing interests + funding statements
- [ ] Classification + keywords entered in submission system

## 10. Final integrity sweep
- [ ] No over-claiming beyond the evidence
- [ ] Significance Statement is broader/plainer than the abstract (not a duplicate)
- [ ] Every figure n / error bar / test defined in legend
- [ ] All accession numbers / DOIs live and correct
