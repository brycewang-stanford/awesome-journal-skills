# 学理文风（ssc-style）

## 触发时机

- 通读后觉得"像政策文件 / 工作汇报 / 教材综述"
- 一段里出现 ≥ 2 个"具有重要意义 / 提供新视角"
- 文风要么口号化，要么纯技术腔，缺学理质感

## 目标语域：宏观视野 + 学理克制

既不流于口号与宏大抒情，也不沦为纯技术报告。每个判断都有学理依据，每个大词都能兑现。

## 对照表（黑名单 → 白名单）

| 问题 | 黑名单 | 白名单 |
|------|--------|--------|
| 空洞价值 | "具有重大的理论价值" | "为理解 X 提供了 Y 框架，修正了 Z 判断" |
| 口号化 | "必须高度重视 / 大力推进" | "这一机制表明，制度设计应在 X 处优先于 Y" |
| 套话贡献 | "丰富了相关研究" | "提出 <概念>，重排了对 X 的既有认识" |
| 炫技 | "本文运用复杂的 XX 方法" | "借助 XX 方法，本文得以分离出 Y 机制" |
| 文献堆砌 | "（A,2020；B,2021；C,2022）" | 按源流/进展/位置分述 |
| 大词空转 | "推动文明进步" | 落到具体机制与边界，少用不可兑现的大词 |

## 行文规范

- **一段一概括**：每段首句是概括句，审稿人读首句即可复原全文逻辑
- 概念全文一致，慎用未界定的大词
- 谦逊与边界：经验研究说"提供了证据支持"，不说"证明了"
- 跨学科稿照顾他学科读者：少用本学科黑话或首次出现即解释

## 自检清单

- [ ] "具有重要意义"类空话命中 = 0
- [ ] 无政策汇报腔（"加强/完善/推进/深化"四件套）
- [ ] 每段首句为概括句
- [ ] 大词都能兑现为具体机制/边界
- [ ] 方法描述不长于思想阐释

## 反模式

- 通篇"重要意义 / 高度重视"
- 用排比大词替代论证
- 方法段长于结论段

## 输出格式

```
【空话命中】X 处：[...]
【政策腔】有 / 无
【段首概括率】X / 总段数
【大词兑现】到位 / 空转 [...]
【下一步】ssc-abstract-keywords
```
