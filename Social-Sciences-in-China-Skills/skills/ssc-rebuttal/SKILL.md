# 外审回复（ssc-rebuttal）

## 触发时机

- 收到《中国社会科学》外审/编辑部修改意见
- 需要逐条回复并对应修改正文

## 顺序：先改正文，后写回复

回复信不能跑在正文修订前面。**先动正文，再写"已如何修改 + 在何处"**，否则空头承诺会被识破。

## 本刊审稿的关注点（据其定位）

- 原创命题是否成立、是否被过度外推
- 理论对话是否到位（经典/流派、跨学科）
- 论证链是否严谨、概念是否清晰
- 思想分量与现实/理论意义是否匹配
- 体例（页下注、引文准确性）

## 逐条回复模板

```
【意见N】<审稿人原话要点>
【回应】<接受/部分接受/审慎商榷——给出学理理由>
【修改】正文第X节第Y段：<具体改动>
【证据/引文】<新增论证/文献/材料>
```

## 三类意见的处理

| 意见类型 | 策略 |
|----------|------|
| 命题/理论质疑 | 优先正面回应：补论证、明确边界，必要时收窄命题（不硬扛） |
| 文献/对话不足 | 补理论脉络与跨学科对话，落到正文，不只在回复信里说 |
| 体例/表述 | 照单全收并修订，回复从简 |

## 礼仪与分寸

- 对编辑/审稿人保持尊重，分歧处"审慎商榷"并给理由，不情绪化
- 真接受就真改，别口头接受、正文不动
- 重大修改在回复信开头做"修改总览"，便于复审定位

## 自检清单

- [ ] 每条意见都有回应 + 对应正文改动位置
- [ ] 命题质疑被正面处理（补证/收窄边界）
- [ ] 理论/跨学科对话的补充落到了正文
- [ ] 体例问题全部修订
- [ ] 语气专业、商榷有据

## 反模式

- 回复信承诺修改，正文未动
- 对命题质疑硬扛或回避，不调整边界
- 把审稿人意见情绪化对待

## 输出格式

```
【意见总数】N　【已逐条回应】N
【修改总览】<3–5 条主要修改>
【命题处理】正面回应 / 收窄边界 / 商榷
【未采纳项】<条目 + 学理理由>
【复审就绪】是 / 否
```
