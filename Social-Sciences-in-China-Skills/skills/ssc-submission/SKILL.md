# 投稿前 preflight（ssc-submission）

## 触发时机

- 正文、命题、文风都定了，准备投稿
- 需要一份"投出去前最后核对"的清单

## 关键事实（详见 resources/journal-profile.md）

- 投稿系统：**https://sscp.portal.founderss.cn/**（亦可经官网 sscp.cssn.cn 入口）
- 注释体例：**页下注（脚注）**，引文出处注遵该刊规范
- 摘要 200–300 字；关键词 3–5；配中英文标题/摘要/关键词
- 主办：中国社会科学院 / 中国社会科学杂志社；月刊

> 体例与字数以官网**最新《投稿须知》**为准。规则会变，投前务必复核，不要凭记忆。

## 体例 checklist

- [ ] 注释统一为**页下注（脚注）**，格式符合该刊规范（作者、文献、出版信息、页码齐全）
- [ ] 引文准确、可核查；直接引用标页码
- [ ] 中英文标题、摘要、关键词齐全且对齐
- [ ] 摘要 200–300 字、关键词 3–5
- [ ] 图表规范、编号连续、有出处
- [ ] 基金项目、作者信息按系统要求填写

## 匿名评审 hygiene

- [ ] 正文/注释/致谢中**去除可识别作者身份**的信息
- [ ] 自引改为第三人称或匿名占位
- [ ] 文件属性（作者名）已清除

## 提交前内容复核（调用其它 skill）

- [ ] 原创命题立得住（`ssc-theory-contribution`）
- [ ] 文献进入理论脉络（`ssc-literature-dialogue`）
- [ ] 论证链无断点（`ssc-argumentation`）
- [ ] 空话/政策腔命中 = 0（`ssc-style`）

## 反模式

- 凭记忆套体例，不查最新《投稿须知》
- 用尾注/文中夹注替代页下注
- 忘清匿名信息就投双盲

## 输出格式

```
【体例】页下注□ 引文□ 图表□ 中英对齐□
【字数】摘要 X 字 / 关键词 X 个
【匿名】已清理 / 待清理 <点>
【系统】sscp.portal.founderss.cn 已确认入口
【内容复核】命题□ 文献□ 论证□ 文风□
【结论】可投 / 待修 <清单>
```
