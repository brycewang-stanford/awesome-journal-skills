# 理论脉络对话（ssc-literature-dialogue）

## 触发时机

- 文献只引近五年实证，没有经典/流派
- 综述写成"张三发现…李四发现…"的流水账
- 评审："缺乏与基础理论的对话"

## 综合刊的文献观

《中国社会科学》看的是**学术脉络感**：你站在哪个理论传统里、回应谁、推进什么。跨学科稿尤其要让相关学科的同行看出"你读过这一脉的根"。

## 三段式综述（替代流水账）

1. **理论源流段**：勾勒该议题的经典传统与主要流派（点到关键作者/著作）
2. **当代进展段**：近年研究在哪几条线上推进，分歧何在
3. **本文位置段**：本文承接哪条线、补哪个缺口、与哪个判断争论

## 经典与本土并重

- 触及**理论源流**（学科奠基作者与代表作），不能只有近期论文
- 跨学科稿：引入相关学科的经典对话对象，避免"自说自话"
- 重视中文学术传统中的本土理论资源，纳入对话而非装饰

## 自检清单

- [ ] 综述进入了理论脉络（有经典/流派，不只近年实证）
- [ ] 按"源流—进展—本文位置"组织，而非按作者罗列
- [ ] 指明了本文介入的具体分歧/缺口
- [ ] 跨学科对话对象到位，相关学科同行能认领
- [ ] 引用为论证服务，无堆砌式并列引注

## 反模式

- "（张三，2020；李四，2021；王五，2022）"式并列，不分述
- 只综述方法/数据演进，回避理论分歧
- 把综述写成"无人反对本文"的铺垫

## 输出格式

```
【理论源流】<传统/流派/关键作者>
【当代分歧】<两三条线及其对立>
【本文位置】承接<…> 补<…> 争论<…>
【脉络感】足 / 不足（缺哪个传统）
【下一步】ssc-argumentation
```
