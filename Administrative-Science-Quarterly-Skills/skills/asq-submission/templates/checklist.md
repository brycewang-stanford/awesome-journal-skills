# ASQ Pre-Submission Self-Check

Verify all volatile specifics (word/abstract limits, formatting details, fees, editorial team)
against the current **Manuscript Submission Guidelines** on ASQ's official page before relying on them.

## 1. Fit & contribution

- [ ] Delivers a *surprising* theoretical insight about organizations (not merely a finding)
- [ ] Contribution is **theoretical**, not methodological
- [ ] Phenomenon sits in organization theory / sociology of organizations
- [ ] The scholarly conversation it joins can be named in one sentence
- [ ] 2–3 recent ASQ articles identified as conversation anchors
- [ ] If purely conceptual with no data: reconsidered (AMR may fit better)

## 2. Theory & argument

- [ ] Process vs. variance form matches the phenomenon
- [ ] A named generative mechanism (actors + structure + logic), not just constructs
- [ ] Boundary conditions stated
- [ ] Deductive: hypotheses derived (not asserted); inductive: process/theory model present

## 3. Methods & evidence

- [ ] Design justified as following from the question
- [ ] Qualitative: theoretical sampling, triangulated sources, trustworthiness safeguards
- [ ] Quantitative: identification strategy explicit; measurement validity addressed
- [ ] Alternative explanations designed/tested against
- [ ] Claims matched to what the evidence can support (no overclaiming)

## 4. Exhibits

- [ ] Qualitative: data-structure figure + data-to-theory/evidence table included
- [ ] Quantitative: descriptives, correlations, main tables complete
- [ ] Process model figure (if process theory) with meaningful arrows
- [ ] Every exhibit self-contained (title, units, notes, significance key)
- [ ] Every exhibit numbered and referenced in text

## 5. Writing & craft

- [ ] Opens with the puzzle, not a literature dump
- [ ] Single theoretical thread runs through every section
- [ ] Process papers written in process voice (not variance-speak)
- [ ] Discussion leads with the contribution and ends on generativity
- [ ] Jargon defined once; prose clean and legible

## 6. References

- [ ] ASQ/SAGE reference style (author–date in text; full list)
- [ ] Every in-text citation in the list and vice versa
- [ ] Recent and canonical organization-theory works both engaged
- [ ] No unresolved "in press"; DOIs/pages complete

## 7. Anonymization (double-blind)

- [ ] Main document free of author-identifying text
- [ ] Self-citations phrased neutrally
- [ ] Acknowledgments/funding on separate title page, not in main file
- [ ] File metadata/properties scrubbed
- [ ] File names contain no author names

## 8. Editorial system & metadata

- [ ] Manuscript type selected (article vs. book review)
- [ ] Title page uploaded as a separate file
- [ ] Cover letter states puzzle, contribution, and ASQ fit
- [ ] Keywords/categories selected accurately
- [ ] Suggested/opposed reviewers handled per system prompts
- [ ] Declarations completed (originality, conflicts, data/ethics as applicable)
- [ ] Confirmed not under review elsewhere
- [ ] Online supplement/appendix uploaded if used
