# Canonical roster — English-NaturalScience-Journal-Skills (100 journals + 1 router)

This file is the **single source of truth** for which venues ship in this bundle,
their slugs (folder name == frontmatter `name`), and a one-line **editorial DNA**
seed per venue so every card is venue-distinctive, not generic. Volatile facts
(impact factors, acceptance rates, ISSNs, word/figure limits, fees) are
deliberately omitted here and deferred to each card's official-submission
checklist, per `_build-spec.md`.

Scope = **top English-language natural-science, clinical/medical, physical, and
formal-science venues** — i.e. the "other 100" complementing the sibling
`English-SocialScience-Journal-Skills` (econ/finance/management 100). The flagship
depth packs in the parent repo (Science, Cell, PNAS, NEJM, The Lancet, Nature)
are intentionally also covered here as breadth cards, mirroring how AER appears in
both a depth pack and the English social-science breadth bundle.

Router slug: `en-natsci-journal-workflow`.
