# 《经济研究》投稿检查清单

## 投稿前格式检查

- [ ] 稿件为DOC或DOCX格式
- [ ] 稿件已完全匿名化处理（正文中无作者、单位、课题等信息）
- [ ] 文件属性中无作者信息

## 作者信息检查

- [ ] 作者总数不超过5位
- [ ] 通讯作者已明确标注
- [ ] 每位作者署名单位不超过2个
- [ ] 每位作者署名资助课题不超过2个
- [ ] 总资助课题数不超过作者总数+2
- [ ] 作者信息真实、准确、完整

## 摘要与关键词检查

- [ ] 中文内容提要约300字
- [ ] 英文梗概约4400字符（含空格）
- [ ] 中文关键词3-8个
- [ ] 英文关键词与中文对应
- [ ] JEL分类号已标注

## 正文结构检查

- [ ] 引言和文献述评部分不少于5000字
- [ ] 结论与政策讨论部分不少于1500字
- [ ] 引言、文献述评、结论各部分表述不简单重复
- [ ] 结论严格演绎自前文理论框架

## 内容质量检查

- [ ] 清晰说明与已有文献（特别是本刊已发表文章）的学术贡献区别
- [ ] 机制分析可靠，非纯粹形式实证
- [ ] 数学表达式的经济含义有文字阐述
- [ ] 政策建议具有可操作性
- [ ] 无空洞的"有意义""有价值"等表述

## 图表与公式检查

- [ ] 正文中表不超过6个
- [ ] 正文中图不超过4个
- [ ] 图表总数不超过8个
- [ ] 数学表达式不超过20个
- [ ] 图表有清晰标题和来源说明

## 参考文献检查

- [ ] 参考文献不超过50条
- [ ] 采用顺序编码制
- [ ] 文中引用格式正确：(作者，年份，页码)
- [ ] 中文文献按姓氏拼音排序在前
- [ ] 外文文献按姓氏拼音排序在后
- [ ] 引用内容真实、完整

## 扩展资料检查

准备以下文件（录用后提交）：
- [ ] 附录文件（Word格式，命名为"论文标题－附加材料（附录）"）
- [ ] 数据文件（与统计软件匹配的数据文件或Excel）
- [ ] 软件代码文件
- [ ] 权利人限制说明（如有）

## 实证研究专项检查

- [ ] 数据来源权威（国泰安、Wind、税收调查数据、CFPS等）
- [ ] 样本选择标准清晰
- [ ] 变量定义清晰明确
- [ ] 识别策略说明清楚（如DID、IV等）
- [ ] 内生性问题有处理
- [ ] 稳健性检验完备
- [ ] 机制检验（中介效应或调节效应）
- [ ] 异质性分析

## 投稿系统检查

- [ ] 在本刊投审稿系统注册并通过审核
- [ ] 使用正确网址：https://erj.ajcass.com/Admin/user/login
- [ ] 系统中作者信息完整准确
- [ ] 稿件是最新版本
- [ ] 仔细检查系统中所显示的相关信息
