# 中国行政管理选题（cpa-topic-selection）

## 触发时机

- 用户说"我想投《中国行政管理》，但选题不确定"
- 题目很宏大（"国家治理现代化研究"），但落不成一个可研究的问题
- 不确定该投《中国行政管理》、《公共管理学报》还是《经济研究》

## 三个硬约束

《中国行政管理》对选题的硬约束：

1. **公共管理理论定位**：仅有"现象 + 数据 / 案例 + 对策"不够，必须能挂上某一支公共管理理论（治理理论、政策过程理论、新公共管理 / 新公共服务、官僚制 / 街头官僚、央地关系等）
2. **可研究的中国治理问题**：宏大议题必须收敛为一个**具体、可回答**的研究问题，并能在第一段说清"这个问题为什么在中国当下值得做"
3. **贡献可枚举**：能拆成 3–4 条独立的、互不重叠的贡献（理论 / 经验 / 方法 / 实践）

## 从宏大议题到可研究问题

公共管理选题最常见的病是"太大"。逐级收敛：

```
领域（国家治理现代化）
  → 议题（基层治理的负担问题）
    → 现象（"指尖上的形式主义"——政务 App 泛滥）
      → 研究问题（自上而下的数字化考核如何
                  通过注意力 / 激励机制
                  转化为基层的形式主义负担？）
```

到"研究问题"层才算可做。判断标准：能用一个疑问句写出来，且能想象用什么证据回答。

## 理论框架自检

写完研究问题后，逐条自检：

- [ ] 本文挂在哪一支公共管理理论上？（治理 / 政策过程 / NPM-NPS / 官僚行为 / 央地关系 / 协同治理）
- [ ] 该理论的代表性经典是哪 2–3 篇 / 部？（如奥斯特罗姆、李普斯基街头官僚、金登多源流、新公共服务等）
- [ ] 本文相对该理论的边际差异是什么？（修正 / 扩展 / 引入中国情境的新条件 / 提供首个经验证据）
- [ ] 中国治理情境是否提供了海外文献没有的研究机会？（如压力型体制、运动式治理、央地结构）

少于 3 个 yes → 选题理论性不够，回到理论文献再读两轮。

## 贡献四句式

```
本文的贡献主要体现在以下方面：
第一，[理论维度]——本文将……纳入[公共管理理论框架]，区别于[已有研究]……。
第二，[经验维度]——本文利用[问卷 / 行政数据 / 案例]提供了……的经验证据，填补……空白。
第三，[机制 / 过程维度]——本文揭示了……治理 / 政策过程机制，回答了"如何发生"的问题。
第四，[实践维度]——本文为[具体治理领域]的制度优化提供了有据可依的启示。
```

不必凑满四条，但**理论 + 经验 / 过程**至少各一条；只有"实践维度"一条 → 像政策咨询报告，不是学术论文。

## 期刊匹配判断

|              | 中国行政管理 | 公共管理学报 | 经济研究 |
|----------|-----------|-----------|--------|
| 学科        | 公共行政 / 治理 | 公共管理（偏方法） | 经济学 |
| 方法        | 定量 + 定性并重 | 偏实证 / 量化 | 定量因果识别 |
| 理论框架    | **公共管理理论必备** | 公共管理理论必备 | 经济学理论 |
| 案例 / 质性 | 一等公民       | 接受       | 较少 |
| 政策含义    | 可操作、面向实践 | 偏理论       | 意义层 |

如果稿子本质是经济学（核心是因果识别 + 经济理论），**不要投《中国行政管理》**。
