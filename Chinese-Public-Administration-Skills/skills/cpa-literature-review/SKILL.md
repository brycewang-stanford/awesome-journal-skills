# 中国行政管理文献综述（cpa-literature-review）

## 触发时机

- 综述缺**公共管理理论文献**（《中国行政管理》审稿人最常挑这一点）
- 综述只有中国经验文献、没有国际公共管理理论坐标，或反之
- 综述写成了"罗列式"而非"对话式"

## 结构规范

《中国行政管理》偏好**三层结构**的综述：

1. **第一层：理论文献**——建立问题的公共管理理论坐标（治理、政策过程、官僚行为、央地关系等经典与近年发展）
2. **第二层：经验 / 实践文献**——梳理已有针对该议题的中国治理经验研究与国际比较研究
3. **第三层：本文位置**——明确指出"本文与现有研究的差异"以及补的空白

### 与经济学期刊综述的差异

《中国行政管理》看重**公共管理理论是否到位**，而非经济学理论或纯计量文献。一篇研究政策执行却不引李普斯基（街头官僚）、马特兰德（模糊—冲突模型）、普雷斯曼-威尔达夫斯基（执行）等执行研究脉络，或研究治理却不引奥斯特罗姆、罗茨 / 治理网络脉络的稿件，会被审稿人挑出。

## 中外文献与新旧配比

经验法则（以官网当年要求为准，不强求固定条数）：

- **公共管理理论文献必须到位**：所属理论分支的经典原始文献，不能只引教科书或综述
- 中外文献**并重**：既要有国际公共管理理论坐标，也要有中国治理本土文献
- 党政政策语境文献**适度纳入**：与议题相关的重要政策文件、权威表述可作为现实背景引用，但不能用政策文件替代学术文献
- 近年文献占比足够，体现对学术前沿的跟踪
- 避免"自引扎堆"（自引宜少）

## 必引避坑清单

- [ ] 是否引用了所属理论分支的**经典原始文献**？（治理 / 政策过程 / 官僚行为脉络）
- [ ] 是否引用了近年公共管理顶刊 / 权威期刊的同主题论文？
- [ ] 是否引用了相关的**中国治理本土研究**？
- [ ] 引用是否覆盖三类：理论、方法、本土经验 / 政策语境？
- [ ] 是否避免了用政策文件堆砌替代学术对话？

## 综述句式模板

```
[作者 (年份)] 基于[数据 / 案例 / 视角]研究了[问题]，认为……。
然而，该研究[局限]，未关注[本文要补的空白]。
与之相比，本文从[理论框架]出发……。
```

每段综述至少出现一次"与之相比 / 区别于 / 本文延伸了 / 本文补充了"。

## 对话式 vs 罗列式

《中国行政管理》偏好**对话式**综述（研究之间相互回应），而非**罗列式**（按作者顺序点名）。

### 错误（罗列式）

> 张三（2020）研究了政策执行，李四（2021）研究了央地关系，王五（2022）研究了基层治理。

### 正确（对话式 — 按主题分组、表明分歧、给出本文位置）

> 政策执行偏差的成因，文献存在两种解释路径：一类强调自上而下的目标模糊与激励扭曲（张三，2020），另一类强调街头官僚的自由裁量（李四，2021）。两者分歧在于偏差的主导动力来自结构还是行动者。少数研究尝试整合两条路径（王五，2022），但其证据局限于单一案例。本文沿这一思路，结合 X 与 Y，进一步考察了 Z。

判断标准：把作者名都隐去后，综述还能读出"研究 A 怎么说、研究 B 怎么反驳、本文怎么定位"，就是对话式。

## 与本文贡献阐述衔接

综述末尾必须导出本文的差异化定位：

> 本文与现有文献的重要区别体现在：
> 第一，[理论框架上的差异]；
> 第二，[研究设计 / 资料上的差异]；
> 第三，[机制或情境上的差异]。

## 反模式

- 把综述写成"教科书式知识介绍"或概念辨析堆砌
- 通篇只有中国经验文献，缺国际公共管理理论坐标（或反之）
- 用大段政策文件原文替代学术文献对话
- 综述里出现 5 个并列引用并排堆砌 —— 拆开按贡献分述
- 引述了治理 / 政策过程的"名词"，却没引相应的经典脉络

## 输出格式

```
【结构】三层 / 两层 / 一层
【理论锚定】到位（已引经典）/ 缺失
【中外配比】中文偏多 / 英文偏多 / 均衡
【政策语境处理】恰当 / 喧宾夺主
【缺漏清单】[...]
【下一步】cpa-identification
```
