# 《中国行政管理》投稿检查清单

> 提示：字数、文献条数、图表数量等具体限制以官网当年"投稿须知 / 稿约"为准。本清单覆盖持久规范。

## 投稿前格式检查

- [ ] 稿件为 DOC 或 DOCX 格式
- [ ] 稿件已完全匿名化处理（正文、文件属性中无作者、单位、课题信息）
- [ ] 章节编号规范：一、二、三 →（一）（二）→ 1、2、3
- [ ] 摘要、关键词、中图分类号齐全（按要求）

## 作者信息检查

- [ ] 通讯作者已明确标注
- [ ] 作者署名单位、资助课题信息真实准确
- [ ] 作者信息只在投稿系统填写，不在正文出现

## 摘要与关键词检查

- [ ] 中文摘要说清"问题—理论框架—方法—发现—启示"四段法
- [ ] 摘要无"具有重要意义"等空泛套话
- [ ] 英文摘要与中文语义对齐
- [ ] 关键词覆盖理论框架、方法、研究对象

## 正文结构检查

- [ ] 有明确的公共管理理论框架（治理 / 政策过程 / NPM-NPS / 官僚行为等）
- [ ] 研究问题具体、可回答（非宏大空泛）
- [ ] 研究设计与问题匹配（定量 / 定性 / 规范）
- [ ] 结论与前文理论框架、发现一致演绎

## 内容质量检查

- [ ] 与已有文献（特别是本领域权威成果）的贡献区别说明清楚
- [ ] 机制 / 政策过程分析可靠，非纯描述
- [ ] 分析部分回扣理论框架（理论不是装饰）
- [ ] 政策含义可操作且能追溯到本文发现
- [ ] 无"强化 / 完善 / 健全 / 推进"四件套套话

## 图表与公式检查

- [ ] 定量主表三线表格式、列数克制、注释含聚类层次
- [ ] 问卷研究有测量 / 信效度表
- [ ] 案例 / 扎根研究有资料来源表 + 编码表（证据链可追溯）
- [ ] 机制 / 过程 / 分析框架图清晰、≥ 300 dpi
- [ ] 图表编号连续，与正文引用对应

## 参考文献检查

- [ ] 公共管理理论经典文献齐全
- [ ] 采用国标著录格式（GB/T 7714，以官网为准）
- [ ] 中外文献按各自格式排列
- [ ] 政策文件引用规范，不与学术文献混排
- [ ] 文中引用与文献列表一一对应

## 研究方法专项检查

定量：
- [ ] 数据来源权威、抽样 / 样本说明清晰
- [ ] 变量定义 / 测量明确，信效度报告（问卷）
- [ ] 识别策略说明（如 DID / IV / 多层模型）、内生性处理
- [ ] 稳健性、机制、异质性分析完备

定性：
- [ ] 案例选择逻辑清晰
- [ ] 资料多源三角验证，证据链可追溯
- [ ] 编码过程透明（扎根 / 内容分析），竞争性解释已排除
- [ ] 分析性概括的外部效度论证

## 研究伦理与投稿系统检查

- [ ] 问卷 / 访谈知情同意，敏感信息脱敏
- [ ] 定性资料去标识（地点 / 受访者匿名）
- [ ] 在官网投稿系统注册并提交
- [ ] "研究类型"选择准确
- [ ] 稿件是最新版本，系统信息完整准确
- [ ] 已核对官网当年字数 / 格式 / 附件要求
