# 投稿前 Preflight（cpa-submission）

## 触发时机

- "明天就要投了"
- 准备点提交按钮前的最后检查
- 不确定投稿系统需要哪些附件

## 重要提示：以官网"投稿须知"为准

字数上限、摘要字数、参考文献条数、图表数量、版面费等**易变数字**请在投稿前到《中国行政管理》官网"投稿须知 / 稿约"核对当年要求。本清单只覆盖**持久的规范**。

## 投稿前 Checklist

### 格式

- [ ] 正文字体 / 行距按官网当年要求（一般宋体，1.5 倍行距，具体以稿约为准）
- [ ] 章节编号：一、二、三（一级）→（一）（二）（三）（二级）→ 1、2、3（三级）
- [ ] 图表编号连续且与正文引用对应
- [ ] 参考文献按国标著录格式（GB/T 7714，具体以官网为准）
- [ ] 摘要、关键词、中图分类号 / JEL（如要求）齐全

### 匿名（匿名评审）

- [ ] 正文中**任何能识别作者身份**的内容已去除
  - 自引写"已有研究（XX，20XX）"，不要写"作者前期研究"
  - 致谢、基金信息**单独**提交，不在正文
  - 作者联系方式只在投稿系统填，不写正文
- [ ] **定性研究的去标识**：访谈对象、调研地点等如涉及保密 / 隐私，作匿名化处理（如"X 省 Y 区""受访者 A"）
- [ ] 文件名不含作者姓名 / 单位；文档属性中无作者信息

### 参考文献

- [ ] 公共管理理论经典文献是否齐全（审稿人易在此挑刺）
- [ ] 中外文献按各自格式排列；中文期刊用全名
- [ ] 每条文献完整：作者、年份、题目、期刊 / 出版社、卷期页
- [ ] 文中每条引用都在文献列表中，列表每条都在文中出现
- [ ] 政策文件引用规范（发文机关、文号、年份），不与学术文献混排

### 研究伦理与资料（公共管理常涉人类被试 / 田野）

- [ ] 问卷 / 访谈是否说明知情同意
- [ ] 敏感信息是否脱敏
- [ ] 数据 / 资料来源合规，遵守授权范围
- [ ] 如涉伦理审查，准备相关说明（按要求）

### 附件

- [ ] 致谢页（单独）
- [ ] 基金资助说明（单独）
- [ ] 作者简介（单独）
- [ ] 数据 / 资料可用性或代码说明（如要求）
- [ ] 利益冲突声明（如要求）

### 查重

- [ ] 知网查重符合期刊要求（保险起见尽量低）
- [ ] 自引比例不过高，必要时说明
- [ ] 工作论文 / 会议论文版本是否已声明

## 投稿系统操作要点

- 通过《中国行政管理》官网在线投稿系统注册并提交；单次上传文件大小有限制，勤刷新
- "研究类型"务必准确选择（实证 / 案例 / 比较 / 规范 / 综述等，按系统选项）
- 推荐 / 回避审稿人按系统要求填写（如有）

## 反模式

- 通宵改完没做匿名检查（尤其定性资料里残留地点 / 人名）
- 参考文献仍是默认 APA 格式，未转为国标
- 图表编号断号 / 图 3 在正文中称"图 4"
- 把政策文件当学术文献混排在参考文献里
- 凭记忆套用"字数上限 / 文献条数"而不查当年稿约

## 输出格式

```
【格式合规】通过 / 待修改：[...]
【匿名合规】通过 / 待修改（含定性去标识）：[...]
【参考文献】理论经典齐全 / 缺；格式国标 / 待转
【研究伦理】已说明 / 不适用 / 待补
【查重】符合要求 / 待降
【附件齐全】是 / 否
【已核对官网当年要求】是 / 否
【下一步】等待外审 / 收到意见 → cpa-rebuttal
```
