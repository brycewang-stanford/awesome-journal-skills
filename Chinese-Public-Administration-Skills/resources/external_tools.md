# 公共管理研究外部工具与资源

本文档整理了向《中国行政管理》投稿公共行政 / 公共政策研究论文时可能需要的外部数据资源、定量与定性分析软件以及辅助工具。公共管理研究方法多元，下面按"定量数据""定性资料""政策文本""分析软件"分类。

> 说明：本清单仅作方法学参考。各数据库的授权范围、各软件的当前版本会变化，使用前请核对官方说明与使用协议。

## 二、定性资料来源

公共管理大量研究采用案例研究、比较案例、过程追踪与扎根理论，资料来源应**多源三角验证**：

| 资料类型 | 具体来源 | 用途 |
|---------|---------|------|
| 政策文件 | 中央 / 地方政府网站、政报、部门规章 | 政策过程、制度变迁 |
| 半结构访谈 | 官员、一线执行人员、服务对象 | 执行逻辑、行动者动机 |
| 实地观察 / 田野笔记 | 政务大厅、社区、街道办 | 治理日常、互动 |
| 档案与会议纪要 | 内部文件（经授权） | 决策过程还原 |
| 新闻报道与媒体报道 | 权威媒体 | 事件时间线、外部佐证 |
| 统计公报与年报 | 政府公开报告 | 背景与量化佐证 |

定性资料须建立**资料清单（chain of evidence）**：每条论断都能追溯到具体资料编号。

## 四、分析软件

### 定量（Stata / Python / R）
- **Stata 17/18**：`reghdfe`（高维固定效应）、`didregress` / `csdid`（交叠 DID）、`ivreghdfe`（工具变量）、`rdrobust`（断点回归）、`xtreg`（面板）、多层模型 `mixed`
- **Python**：`pandas`、`statsmodels`、`linearmodels`（面板 / IV）、`scikit-learn`、`matplotlib` / `seaborn`
- **R**：`fixest`（高维固定效应）、`did`（Callaway-Sant'Anna）、`lme4`（多层 / HLM）、`MatchIt`（匹配）、`rdrobust`

### 多层 / 嵌套结构
公共管理数据常呈"个体—组织—地区"嵌套结构：
- HLM 软件、Stata `mixed` / `meglm`、R `lme4` / `brms`

### 定性分析（CAQDAS）
| 软件 | 说明 |
|------|------|
| NVivo | 主流质性编码、节点管理、矩阵查询 |
| ATLAS.ti | 编码、网络视图、扎根理论支持 |
| MAXQDA | 混合方法、定量定性整合 |
| Dedoose | 在线协作、混合方法 |

### 定性比较分析（QCA）
- `fsQCA` 软件、R `QCA` 包、Tosmana —— 适用条件组合 / 因果复杂性研究

### 社会网络分析
- UCINET、Gephi、R `igraph` / `statnet` —— 适用治理网络、政策网络

## 五、混合方法设计辅助

- **三角验证矩阵**：定量发现 × 定性证据交叉印证
- **嵌入式 / 解释性序列设计**：先定量后定性解释，或先定性建模后定量检验
- **过程追踪检验**：箍式检验（hoop test）、冒烟枪检验（smoking-gun test）的证据强度判断

## 八、注意事项

1. **方法服务于问题**：先想清研究问题，再选定量 / 定性 / 混合方法，不要为方法而方法
2. **资料合规与伦理**：访谈须知情同意，敏感信息脱敏；行政数据遵守授权范围
3. **可追溯性**：定量保留完整 do / 代码；定性建立资料编号与编码簿，便于审稿核查
4. **匿名处理**：投稿前确保正文、资料、文件属性中无作者与受访者可识别信息
5. **以官网为准**：格式、字数、附件要求随年度变化，提交前核对当年"投稿须知"
