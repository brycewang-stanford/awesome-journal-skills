# Science Submission Preflight Checklist

> Copy this file, tick every box, and resolve all blocking items before upload.
> Targets reflect typical Science guidelines — confirm exact caps in the current Author Guidelines.

## 1. Significance & framing
- [ ] Broad significance / general interest established (sci-fit rung ≥ 3)
- [ ] One-sentence advance demonstrated (not asserted)
- [ ] Title declarative, specific, < ~90 characters

## 2. Format & length
- [ ] Format: ☐ Report (~2,500 words, ≤4 items)  ☐ Research Article (~4,500 words, ≤6 items)
- [ ] Main-text word count within budget: ______ words
- [ ] Methods in Supplementary Materials (for Reports)
- [ ] Results ordered by argument; claim-first paragraphs

## 3. Abstract & one-sentence summary
- [ ] One-sentence summary ≤ ~125 characters: ______ chars
- [ ] Abstract ≤ ~125 words: ______ words
- [ ] No subheadings / no citations in abstract
- [ ] ≥ 1 quantified headline result in abstract

## 4. Figures & display items
- [ ] Item count within budget: ______
- [ ] Sized to 5.5 / 12 / 18 cm; min font ≥ 6 pt
- [ ] Data shown (points + n); error bars defined (SD/SEM/CI)
- [ ] Scale bars on images; uncropped key blots + source data retained
- [ ] Colorblind-safe palette; meaning not by color alone
- [ ] Each legend: title sentence + per-panel + statistics

## 5. Statistics & reproducibility
- [ ] Per claim: effect size + 95% CI + n + test + assumptions
- [ ] Biological vs technical replication distinguished
- [ ] Sample-size rationale / power stated
- [ ] Randomization & blinding reported or justified
- [ ] Multiple-comparison correction where needed
- [ ] Analysis code public + archived with DOI

## 6. Data, materials, ethics
- [ ] Data deposited; accession numbers / DOIs in hand: ______
- [ ] Data-availability statement compliant (no "on request" for primary data)
- [ ] Materials/reagent sharing plan (Addgene / MTA)
- [ ] Ethics: IRB ☐  IACUC ☐  field/biodiversity permits ☐  N/A ☐

## 7. References
- [ ] Science numbered style, ordered by appearance
- [ ] One continuous list (main + SM)
- [ ] Full author lists; journal abbreviations
- [ ] All in-text reference numbers resolve; no gaps/duplicates

## 8. Required files & metadata
- [ ] Main text (title, one-sentence summary, abstract, references)
- [ ] Figures at required resolution + figure legends
- [ ] Supplementary Materials (Materials and Methods, supp figs/tables, captions)
- [ ] Cover letter (significance-forward)
- [ ] Authors, affiliations, ORCIDs, corresponding author
- [ ] CRediT author contributions
- [ ] Competing interests + funding statements
- [ ] Suggested / opposed reviewers (if used)

## 9. Final integrity sweep
- [ ] No over-claiming beyond the evidence
- [ ] Every figure n / error bar / test defined in legend
- [ ] All accession numbers / DOIs live and correct
- [ ] Blinding convention matches the journal (Science typically single-blind)
