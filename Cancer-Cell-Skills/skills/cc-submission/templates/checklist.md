# Cancer Cell (Cell Press) Pre-Submission Checklist

> Durable norms only. Verify all volatile specifics (word/figure limits, fees, required sections, portal URL) on the official Cancer Cell / Cell Press author pages before submitting.

## Scope & contribution

- [ ] One sentence states the molecular mechanism (molecule/axis → cancer phenotype)
- [ ] Mechanism validated across orthogonal systems (cells + in vivo + ideally human/patient data)
- [ ] Translational relevance is explicit and matched to the evidence level
- [ ] Advance over the closest prior work is specific (new node / context / in vivo proof)
- [ ] Study is hypothesis-driven, not purely descriptive

## Manuscript files & format

- [ ] Manuscript in required format (DOC/DOCX or PDF), with line and page numbers
- [ ] Figures at required resolution/format; numbered and cited in order
- [ ] Supplemental items packaged and referenced in text
- [ ] Graphical abstract file at required size/resolution (if used)
- [ ] Cover letter prepared
- [ ] Highlights and eTOC blurb included

## Front matter

- [ ] Summary leads with the discovery; within current word limit; no hype words
- [ ] Summary states orthogonal validation and a calibrated significance line
- [ ] Highlights are results (not aims/methods), within count and length
- [ ] eTOC blurb is broad-audience, present tense, accurate
- [ ] Graphical abstract conveys mechanism → consequence and matches the data

## STAR Methods & Key Resources Table

- [ ] STAR Methods headings used; Key Resources Table (KRT) at the top
- [ ] Antibodies: host, clone, vendor, catalog #, RRID, application/dilution
- [ ] Cell lines: source, STR authentication, RRID (Cellosaurus); mycoplasma-negative stated
- [ ] Commonly misidentified lines flagged (ICLAC) and use justified
- [ ] Organisms/strains, plasmids (Addgene/RRID), software (version + RRID) listed
- [ ] Lead contact and materials-availability statements present

## Animal rigor (if applicable)

- [ ] Species, strain, sex, age, source, housing reported
- [ ] Sample size justified; randomization described; outcome assessment blinded
- [ ] Inclusion/exclusion criteria and humane endpoints pre-defined and reported
- [ ] IACUC approval with protocol number and institution

## Human samples (if applicable)

- [ ] IRB/ethics approval (institution + protocol)
- [ ] Informed consent obtained (or documented waiver)
- [ ] Declaration of Helsinki compliance affirmed
- [ ] Controlled-access deposition for patient-identifiable genomics (dbGaP/EGA)
- [ ] Trial registration number, if samples/outcomes came from a registered trial

## Statistics

- [ ] `n` defined per panel as biological replicates; one biological unit specified
- [ ] No pseudo-replication (technical replicates not counted as `n`)
- [ ] Tests named and matched to design; assumptions checked
- [ ] Longitudinal/tumor-growth data analyzed with appropriate model (not serial t-tests)
- [ ] Multiple comparisons corrected; FDR for omics
- [ ] Error bars defined (SD/SEM/CI) in every legend; exact p where feasible
- [ ] Statistical software and versions stated

## Figures & image integrity

- [ ] Each representative image paired with replicate-level quantification
- [ ] Data points shown for small-n panels (superplots)
- [ ] Uncropped/full-length blots in supplement; MW markers visible
- [ ] No undisclosed splicing; image adjustments linear and whole-image
- [ ] No duplicated/reused panels; scale bars on all micrographs
- [ ] In vivo / human validation panels present where claims require them

## Data & code availability

- [ ] All newly generated datasets deposited; accessions listed and matching the KRT
- [ ] GEO/SRA (sequencing), PRIDE (proteomics), PDB/EMDB (structures), etc., as applicable
- [ ] Reviewer access tokens prepared for embargoed records
- [ ] Custom code deposited in a citable repository with a DOI
- [ ] No data type left as "available on request" that should be deposited

## Declarations & authorship

- [ ] Competing-interests declaration complete
- [ ] Funding statement complete
- [ ] Author contributions (CRediT-style) stated
- [ ] All author affiliations correct; corresponding author and ORCIDs set

## Consistency & portal

- [ ] Gene/protein nomenclature consistent (HGNC/MGI; gene italics; species capitalization)
- [ ] Every in-text figure/table/reference citation resolves; reference style correct
- [ ] Claims match evidence (no therapeutic claim without in vivo/human support)
- [ ] Suggested/excluded reviewers ready (conflict-free)
- [ ] Submitted through the current Cell Press portal; all metadata fields complete
