# JAMA Pre-Submission Checklist

> Verify all word counts, reference caps, and exhibit limits on JAMA's current
> Instructions for Authors page. The items below are durable requirements.

## 1. Article type & limits

- [ ] Correct JAMA article type selected (Original Investigation, Review, Research Letter, etc.)
- [ ] Manuscript within the word limit for that type (verify number)
- [ ] References within the cap for that type (verify number)
- [ ] Figures + tables within the exhibit limit (verify number)
- [ ] Line numbers and page numbers present

## 2. Structured abstract & Key Points

- [ ] Abstract uses JAMA headings: Importance; Objective; Design, Setting, and Participants;
      Interventions/Exposures; Main Outcomes and Measures; Results; Conclusions and Relevance
- [ ] Trial Registration line present (trials) / PROSPERO (reviews)
- [ ] Primary result reported with effect size + 95% CI
- [ ] Abstract within word ceiling (verify, commonly ~350 words)
- [ ] Key Points box present: Question / Findings / Meaning
- [ ] Abstract, Key Points, and body are numerically consistent

## 3. Reporting standard

- [ ] Correct EQUATOR guideline used (CONSORT / STROBE / PRISMA / STARD / other)
- [ ] Completed checklist file with page/line locations for each item
- [ ] Required flow diagram prepared (CONSORT/PRISMA/STARD)

## 4. Statistics

- [ ] Single pre-specified primary outcome; secondaries labeled
- [ ] All estimates reported with 95% CIs (not p-values alone)
- [ ] Intention-to-treat is the primary analysis (RCT)
- [ ] Multiplicity handled with a pre-specified plan
- [ ] Missing-data method stated + sensitivity analysis
- [ ] Absolute and relative effects both reported

## 5. Required statements

- [ ] Trial/protocol registration (registry + identifier)
- [ ] IRB/ethics approval + Declaration of Helsinki
- [ ] Informed-consent statement (or justified waiver)
- [ ] Consent for publication of identifiable individuals
- [ ] ICMJE authorship + contributorship statement
- [ ] Conflict-of-interest disclosures for all authors
- [ ] Funding source + role of the funder/sponsor
- [ ] Data-sharing statement
- [ ] Access-to-data / statistical-responsibility statement (trials)

## 6. Files to upload

- [ ] Main manuscript (structured abstract, Key Points, IMRaD, references)
- [ ] EQUATOR checklist file
- [ ] Flow diagram figure
- [ ] Trial protocol + statistical analysis plan (trials)
- [ ] Cover letter
- [ ] ICMJE COI forms for each author
- [ ] Tables/figures to the journal's file specs (format, resolution)
- [ ] Online-only supplement (if applicable)

## 7. Style & integrity

- [ ] AMA Manual of Style for numbers, units, abbreviations, references
- [ ] Person-first, non-stigmatizing language; race/sex reported per guidance
- [ ] No spin; conclusions bounded by the primary outcome
- [ ] Original work, not under consideration elsewhere
- [ ] Preprint / prior presentation disclosed
- [ ] Plagiarism / text-overlap self-checked

## 8. Portal

- [ ] Registered in JAMA's submission system (verify current URL on the journal site)
- [ ] Author order, corresponding author, and ORCIDs correct
- [ ] Suggested / non-preferred reviewers entered (if allowed)
- [ ] Article type and category selected correctly
- [ ] Latest manuscript version uploaded
