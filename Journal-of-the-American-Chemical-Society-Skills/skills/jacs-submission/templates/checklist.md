# JACS Pre-Submission Checklist

A mechanical, section-by-section gate before uploading to ACS Paragon Plus.
Confirm all volatile specifics (formats, limits, required files, policies) on the
current JACS author page; this list captures durable norms.

## 1. Scope and format

- [ ] The advance is significant and of broad interest across chemistry
- [ ] An honest "why JACS, not a specialized ACS journal?" answer is in hand
- [ ] Article vs Communication chosen to match the manuscript and current norms

## 2. Files and system

- [ ] Manuscript in the ACS Word or LaTeX template
- [ ] Separate Supporting Information PDF
- [ ] CIF files for all crystal structures; CCDC numbers obtained
- [ ] Table of Contents (TOC) / abstract graphic per ACS rules
- [ ] Figures at ACS-specified resolution/format
- [ ] Registered ACS Paragon Plus account; correct journal selected

## 3. Front matter

- [ ] Title states the chemistry (not "studies toward")
- [ ] Author list, affiliations, corresponding author(s) correct
- [ ] ORCID linked as required
- [ ] Abstract: problem → advance → key number → significance
- [ ] TOC graphic encodes the central advance

## 4. Characterization and purity (per new compound)

- [ ] ¹H NMR with solvent, field, δ, multiplicity, J, integration
- [ ] ¹³C{¹H} NMR (and ¹⁹F/³¹P/etc. as relevant)
- [ ] HRMS: ionization mode, calcd vs found m/z
- [ ] IR diagnostic bands where informative
- [ ] Purity evidence: EA (calcd vs found) or HPLC/GC, plus clean NMR
- [ ] mp / [α]D / UV-vis as appropriate to the compound class
- [ ] Copies of key spectra in SI, legible, labeled by compound number

## 5. Stereochemistry (if applicable)

- [ ] ee by chiral HPLC/GC/SFC (method + racemate trace)
- [ ] dr by NMR/HPLC with method stated
- [ ] Absolute configuration assigned with method stated

## 6. Crystallography (if applicable)

- [ ] Single-crystal X-ray for structurally pivotal claims
- [ ] CIF deposited; CCDC number cited in text and SI
- [ ] Crystal/refinement table present
- [ ] checkCIF/PLATON alerts resolved or justified

## 7. Mechanism / catalysis (if applicable)

- [ ] Scope table honest (failures noted)
- [ ] Control experiments included
- [ ] Mechanistic claims supported by kinetics / KIE / labeling / intermediates
- [ ] Catalyst loading, TON/TOF, reproducibility reported

## 8. Computation (if applicable)

- [ ] Functional, basis set/pseudopotentials, solvation, software + version
- [ ] Stationary points characterized (TS = one imaginary frequency)
- [ ] Optimized coordinates (xyz) in SI

## 9. Writing and references

- [ ] ACS numbered citation style, references in order, abbreviated titles
- [ ] Compound numbering consistent across text, schemes, and SI
- [ ] SI units, italic variables, correct spacing (e.g., 5 mol %, 80 °C)
- [ ] Booster words replaced with numbers/facts

## 10. Disclosures and integrity

- [ ] Cover letter (advance + fit) attached
- [ ] Conflict-of-interest statement per ACS policy
- [ ] Funding / acknowledgments handled
- [ ] Data availability statement present
- [ ] Preprint / prior / companion-paper disclosure if applicable
- [ ] Safety hazards disclosed
- [ ] No undisclosed image manipulation

## 11. Reviewers

- [ ] Suggested reviewers: independent, covering needed expertise + breadth
- [ ] Excluded reviewers (if any) with brief justification

## 12. Final proof

- [ ] Paragon Plus proof PDF renders all figures/schemes/Greek/math correctly
- [ ] File sizes within portal limits
- [ ] Manuscript is the latest version
