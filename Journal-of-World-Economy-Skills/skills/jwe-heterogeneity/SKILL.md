# 异质性分析（jwe-heterogeneity）

> 《世界经济》（IWEP + 中国世界经济学会主办，月刊）偏好**国际经济学意义上的切分**：暴露度（进口渗透/出口依赖/GVC 联系/汇率敞口）、贸易类型（一般 vs 加工贸易）、贸易边际（扩展 vs 集约）、目的国/伙伴。东中西、国企民企这类通用维度在本刊价值有限——它们换到任何国内刊都成立，立不住本刊的开放属性。基础事实见 `resources/journal-profile.md`。

## 触发时机

- 异质性只切了东中西 / 国企民企，缺国际经济学意义的维度
- 主效应 + 渠道已立，需要刻画"对谁更强"
- 审稿人会问"效应在不同暴露 / 贸易类型间是否一致"

## 开放经济专属的切分维度（优先这些）

| 维度 | 切分方式 | 为什么有意义 |
|------|----------|--------------|
| **暴露度** | 高 vs 低进口渗透 / 出口依赖 / GVC 联系 / 汇率敞口 | 直接检验冲击经暴露通道传导 |
| **贸易类型** | 一般贸易 vs 加工贸易；中间品 vs 最终品出口商 | 加工贸易对关税 / 汇率响应不同 |
| **企业属性** | 内资 vs 外资；高 vs 低生产率（Melitz 含义） | 异质性企业理论可检验含义 |
| **目的国 / 伙伴** | 发达 vs 发展中；签 vs 未签协定国；远 vs 近 | 引力 / 协定效应的异质性 |
| **贸易边际** | 扩展边际（进入退出）vs 集约边际（量价） | 区分谁在调整 |
| **GVC 位置** | 上游 vs 下游；高 vs 低国内增加值率 | 全球生产网络中的位置差异 |

## 做实异质性（而非堆回归）

- 每个切分都要有**理论预期**：为什么这一组应更强 / 更弱
- 用交互项 + 系数差异检验，别只贴两张分样本表
- 异质性方向应与主渠道一致（高暴露组效应更大 → 支持该渠道）
- 谨防"切到显著为止"的数据探查

## 自检清单

- [ ] 至少一个维度是国际经济学专属（暴露 / 贸易类型 / GVC / 目的国）
- [ ] 每个切分有事前理论预期
- [ ] 报告系数差异的统计检验（交互项 / 组间检验）
- [ ] 异质性方向与主渠道自洽
- [ ] 没有把异质性当"凑显著"的捞针

## 反模式

- 只切东中西 / 大小企业等通用维度
- 切了一堆维度但无理论解释
- 分样本显著差异不做组间检验
- 异质性方向与机制矛盾却不解释

## 输出格式

```
【维度】暴露度 / 贸易类型 / 企业 / 目的国 / 贸易边际 / GVC位置
【理论预期】<每维度一句>
【差异检验】交互项□ 组间检验□
【与渠道一致性】是 / 否 <说明>
【下一步】jwe-tables-figures
```
