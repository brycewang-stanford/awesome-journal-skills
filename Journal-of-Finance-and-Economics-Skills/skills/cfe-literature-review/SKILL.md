# 财经研究文献综述（cfe-literature-review）

## 触发时机

- 综述缺中文经典，或全部是英文文献
- 综述缺**理论文献**（《财经研究》审稿人常挑这一点）
- 综述写成了"罗列式"而非"对话式"
- 没有与近 3 年同主题中文文献（含本刊）对话

## 结构规范

《财经研究》偏好**三层结构**的综述：

1. **第一层：理论文献**（英文为主，少量经典中文）——建立问题的理论坐标
2. **第二层：实证文献**（中英并重）——梳理已有实证发现与分歧
3. **第三层：本文位置**——明确指出"本文与现有研究的差异"

### 与纯理论期刊的差异

《财经研究》是综合性财经期刊，综述既要**理论文献到位**，又要**与同领域中文实证文献充分对话**。一篇按方向归属（如公司财务、公共财政、产业组织）该引的经典理论或经典实证若缺位，会被审稿人挑出"理论定位不清"。

## 中英文献配比

经验法则（具体以稿件方向为准）：

- 总引文约 50–100 条
- 中文文献占比 **30%–50%**（不能 <20%）
- **理论文献占比 ≥ 20%**（不是实证应用，是理论推导或综述）
- 五年内文献占比 **≥ 40%**
- 高水平期刊文献占比 **≥ 35%**（顶级英文期刊 + 经济研究 / 管理世界 / 经济学（季刊）/ 财经研究本刊等）

## 必引避坑清单

- [ ] 是否引用了所属理论分支的**经典理论文献**？（不是教科书，而是原始 paper）
- [ ] 是否引用了近 3 年高水平期刊的同主题论文？
- [ ] 是否引用了相关的**中文经典 / 综述**？（含《财经研究》同主题近文）
- [ ] 引用是否覆盖三类：理论、方法、本土实证？
- [ ] 是否避免了"自引扎堆"？（自引 ≤ 3 条）

## 综述句式模板

```
[作者 (年份)] 利用[数据 / 方法]研究了[问题]，发现……。
然而，该研究[局限]，未考虑[本文要补的空白]。
与之相比，本文……。
```

每段综述至少出现一次"与之相比 / 区别于 / 本文延伸了 / 本文补充了"。

## 对话式 vs 罗列式

《财经研究》偏好**对话式**综述（研究之间相互回应），而非**罗列式**（按作者顺序点名）。

### 错误（罗列式）

> 张三（2020）研究了减税政策，李四（2021）研究了企业投资，王五（2022）研究了融资约束。

### 正确（对话式 — 按主题分组、表明分歧、给出本文位置）

> 减税政策如何影响企业投资是近年的研究热点（张三，2020；李四，2021），但这类研究多假设融资约束外生，对减税通过缓解融资约束影响投资的渠道关注有限。少数研究尝试从融资约束切入（王五，2022），但其数据局限于上市公司大样本。本文沿这一思路，使用工企与税收调查匹配数据，进一步考察了非上市中小企业的投资响应。

判断标准：把作者名都隐去后，综述还能读出"研究 A 怎么说、研究 B 怎么反驳、本文怎么定位"，就是对话式。

## 与本文贡献阐述衔接

综述末尾必须导出本文的差异化定位，模板：

> 本文与现有文献的重要区别体现在以下几个方面：
> 第一，[识别策略上的差异]；
> 第二，[数据或样本上的差异]；
> 第三，[机制或异质性上的差异]。
> 与《财经研究》近期发表的 XXX（作者，202X）一文相比，本文 ……

**与本刊已发表文章对比**是审稿人的关注点——综述若不提近 3 年本刊同主题文章，会被视为"未充分对话"。

## 反模式

- 把综述写成"教科书式知识介绍"
- 综述完全围绕实证，无理论文献
- 中文文献只引一两条凑数
- 综述里出现 [27, 28, 29, 30, 31] 这种 5 个并列引用 —— 拆开按贡献分述
- 自引超过 5 条

## 输出格式

```
【结构】三层 / 两层 / 一层
【中英配比】中文 X% / 英文 Y%
【理论文献占比】X%
【五年内文献比例】X%
【高水平期刊比例】X%
【本刊同主题对话】有 / 无
【缺漏清单】[...]
【下一步】cfe-identification
```
