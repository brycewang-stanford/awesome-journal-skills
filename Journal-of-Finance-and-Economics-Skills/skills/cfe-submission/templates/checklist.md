# 《财经研究》投稿检查清单

> 说明：以下涉及的具体字数、条数、份数等数字，请以官网（https://cjyj.sufe.edu.cn/）"投稿须知"当年要求为准。本清单给出需要核对的**项目类别与持久规范**。

## 一、投稿前格式检查

- [ ] 稿件为 DOC 或 DOCX 格式
- [ ] 稿件已完全匿名化处理（正文中无作者、单位、课题等信息）
- [ ] Word 文件属性 / 元数据中无作者信息
- [ ] 章节编号规范：一、二、三 →（一）（二）→ 1、2、3
- [ ] 公式独立成行、居中、右侧带编号

## 二、作者信息检查

- [ ] 作者人数与署名单位数符合官网要求
- [ ] 通讯作者已明确标注
- [ ] 基金资助课题数符合官网要求
- [ ] 作者信息真实、准确、完整
- [ ] 致谢、基金、作者简介**单独**提交，不出现在正文

## 三、摘要与关键词检查

- [ ] 中文摘要字数符合官网要求
- [ ] 英文摘要字数 / 字符数符合官网要求
- [ ] 中英文摘要语义逐句对齐
- [ ] 中文关键词 + 英文关键词对应
- [ ] JEL 分类号已标注

## 四、正文结构检查

- [ ] 引言点明了**中国现实问题**与研究价值
- [ ] 文献述评为对话式、含理论文献、与本刊近文对话
- [ ] 引言列出 3–5 条边际贡献
- [ ] 结论严格演绎自前文理论框架与实证证据
- [ ] 引言、文献述评、结论各部分不简单重复

## 五、内容质量检查

- [ ] 清晰说明与已有文献（特别是本刊已发表文章）的学术贡献区别
- [ ] 识别策略清楚（DID / IV / RDD / PSM-DID / DML 等）并回应内生性
- [ ] 机制分析可靠，非纯粹形式实证
- [ ] 异质性分析至少 3 个维度且有理论指引
- [ ] 数学表达式的经济含义有文字阐述
- [ ] 政策含义为意义层、回扣中国现实
- [ ] 无空洞的"有意义""有价值"等表述

## 六、图表与公式检查

- [ ] 主表列数克制（≤ 6 列为宜）
- [ ] 表为三线表、无竖线
- [ ] 注释含聚类层次与显著性符号定义
- [ ] 变量定义表给出计算公式 + 点名数据来源
- [ ] 图形 ≥ 300 dpi、配色克制
- [ ] 图表有清晰标题和来源说明、编号连续

## 七、参考文献检查

- [ ] 中文文献与英文文献分别按官网格式排列
- [ ] 中文期刊用全名（不写缩写）
- [ ] 文中引用格式正确
- [ ] 文中引用与文末列表一一对应
- [ ] 理论文献齐全、自引比例合理
- [ ] 引用内容真实、完整

## 八、实证研究专项检查

- [ ] 数据来源权威且点名（国泰安 / Wind / CNRDS / 工企 / 海关 / 税收调查 / CFPS 等）
- [ ] 样本选择标准清晰
- [ ] 变量定义清晰明确（含计算公式）
- [ ] 缩尾 / 异常值处理已说明
- [ ] 平行趋势 / 弱工具 / 平衡性等设计相应检验已做
- [ ] 安慰剂检验完备
- [ ] 稳健性检验完备
- [ ] 机制检验（中介 / 替换 Y / 子样本）
- [ ] 异质性分析含组间系数差异检验

## 九、投稿系统检查

- [ ] 在《财经研究》官网投稿系统注册并按提示完成
- [ ] 使用正确网址：https://cjyj.sufe.edu.cn/
- [ ] 系统中作者信息完整准确
- [ ] 稿件是最新版本
- [ ] 推荐 / 回避审稿人按系统要求填写
- [ ] 仔细检查系统中所显示的相关信息

## 十、扩展资料（录用后按要求提交）

- [ ] 附录文件（Word 格式）
- [ ] 数据文件（与统计软件匹配或 Excel / CSV）
- [ ] 软件代码文件（.do / .py / .R）
- [ ] 数据权利人限制说明（如有）
