# 《管理世界》复现包目录骨架

这个骨架面向中文经管实证论文，特别是使用 CSMAR / Wind / RESSET / iFinD / 中经网、统计年鉴、企业年报、政策文本或受限微观数据的稿件。原则是：不重新分发受限原始数据，但必须让编辑、审稿人或读者理解每个变量从哪里来、如何清洗、如何复现论文表图。

## 推荐目录

```text
replication-package/
├── README.md
├── LICENSE
├── run_all.do
├── code/
│   ├── 00_globals.do
│   ├── 01_clean.do
│   ├── 02_construct_variables.do
│   ├── 03_main_tables.do
│   ├── 04_robustness.do
│   └── 05_figures.do
├── data/
│   ├── README.md
│   ├── raw/            # 不提交受限原始数据；只放公开数据或占位说明
│   ├── intermediate/   # 可由脚本生成，通常不手工修改
│   └── analysis/       # 主回归样本；若受限则提供脱敏/合成样本
├── docs/
│   ├── codebook-template.csv
│   ├── data-acquisition-guide.md
│   └── table-figure-crosswalk.csv
└── output/
    ├── tables/
    └── figures/
```

## README 必写内容

- 论文题目、作者、联系邮箱、复现包版本和最后更新日期。
- 软件版本：Stata / R / Python / LaTeX 及关键包版本。
- 数据来源清单：数据库、模块、表名、字段名、版本、获取日期、是否受限。
- 运行顺序：用 `run_all.do` 或等价总脚本串联全部步骤。
- 产出对照：论文每张表、每张图对应的脚本和输出文件。
- 受限数据说明：不可重分发原因、如何申请访问、可提供的脱敏或合成样本。
- 随机性控制：随机种子、抽样规则、机器学习/匹配/Bootstrap 设置。
- 常见失败处理：缺包、路径、编码、权限、外部字体或图表导出依赖。

## 受限数据处理

不要上传违反授权协议的原始数据。对于商业数据库和政府/调查微观数据，至少提供：

- 字段级定位：数据库 / 模块 / 表名 / 字段名。
- 数据获取指引：订阅机构、申请页面、联系人或官方入口、预计周期。
- 可运行样本：脱敏样本、合成样本或最小公开样本，用于冒烟测试代码。
- 变量构造说明书：每个变量的公式、过滤条件、缺失值处理、单位和取值范围。
