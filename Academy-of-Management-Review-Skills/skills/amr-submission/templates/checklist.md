# AMR Pre-Submission Checklist (Academy of Management Review)

AMR is a **theory-only** journal: no data, no measures, no statistical tests, no
data-availability statement. This checklist reflects that. Verify all volatile specifics
(length, abstract limit, fees, current editorial requirements) on the official AMR /
Academy of Management author page before submitting.

## 1. Scope & fit

- [ ] The manuscript builds NEW theory (new construct / process model / reconceptualization)
- [ ] It enters one specific conversation and both challenges AND extends it
- [ ] There is no dataset, no measure, no estimation, no results section
- [ ] The "before → after" theoretical contribution can be stated in one sentence
- [ ] Confirmed against the current AMR aims & scope

## 2. Format

- [ ] Double-spaced, standard margins, 12-pt font (verify current AOM spec)
- [ ] Separate title page (kept out of the anonymized body)
- [ ] Abstract within the current word limit (verify)
- [ ] Manuscript within the journal's length expectation (verify; AMR runs full-length)
- [ ] Page/line numbering as required by the portal
- [ ] Figures meet AOM resolution and format requirements

## 3. Anonymization (double-anonymous review)

- [ ] No author names, affiliations, or identifying phrases in the body
- [ ] Self-citations neutral ("prior research (Author, 2019)"), never "our earlier study"
- [ ] Acknowledgments and funding identifiers removed (added after acceptance)
- [ ] Document file properties scrubbed of author metadata
- [ ] File names free of author/institution names

## 4. Abstract & front matter

- [ ] Abstract states the puzzle, the theoretical move, and the contribution
- [ ] Keywords selected per the journal's list/conventions
- [ ] No data/results language in the abstract (there are none)

## 5. Theory quality

- [ ] Every construct is defined, domain-bounded, and distinct from neighbors
- [ ] Assumptions stated explicitly
- [ ] Each proposition is preceded by an explicit logical argument and a mechanism
- [ ] Boundary conditions stated as part of the theory
- [ ] Alternative explanations addressed; at least one disconfirming case considered
- [ ] Propositions numbered and matched to the conceptual figure

## 6. Figures (conceptual only)

- [ ] Each figure is a process model / typology / multi-level / relational model — never a data plot
- [ ] Every box is a defined construct; every arrow maps to a proposition or stated mechanism
- [ ] A propositions table accompanies papers with many propositions
- [ ] Figure labels and numbering match the text exactly

## 7. References

- [ ] AOM author-date in-text citations; full reference list per the AOM Style Guide
- [ ] Every in-text citation is in the list and vice versa
- [ ] The recent, live conversation is cited (not only seminal anchors)
- [ ] All references are real and accurately represented

## 8. Ethics & portal (ScholarOne)

- [ ] Original, unpublished, not under review elsewhere (AOM Code of Ethics)
- [ ] Authorship and order agreed by all authors; each made a genuine contribution
- [ ] Prior conference/working-paper version disclosed if required
- [ ] Similarity check run; self-overlap disclosed
- [ ] Registered in the AMR ScholarOne portal; correct article type selected
- [ ] Files staged: anonymized manuscript + separate title page + any required forms
- [ ] Within portal file-size and file-type limits
