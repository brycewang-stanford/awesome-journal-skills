# External Tools & Resources for AMR Theory-Building

The Academy of Management Review (AMR) publishes **conceptual, theory-building articles
only**. There are **no datasets, no estimation, no results section**. So this resource
list is deliberately different from an empirical journal's: it centers on tools for
*finding the theoretical conversation*, *managing a foundational literature*, *drawing
conceptual figures*, and *checking the logic of an argument* — not on statistical
software or data vendors.

> Volatile specifics (editorial team, exact length limits, fees) change. Always verify
> against the official AMR / Academy of Management author pages before submission.

## 1. Locating the theoretical conversation

| Tool | Use |
|------|-----|
| Web of Science / Scopus | Citation-trail mapping: who cites the construct you want to extend, and where the conversation has stalled |
| Google Scholar | "Cited by" forward search to find the most recent theoretical critiques of a construct |
| AOM journals (AMR, AMJ, AMD, AOM Annals, AMP) | The home conversation — AMR theory almost always engages an existing AMR/AMJ stream |
| Connected Papers / Litmaps | Visual citation graphs to see the structure of a theoretical domain and spot the gap |
| AOM Annals review articles | Authoritative state-of-the-field summaries; a good Annals piece tells you where a theory is exhausted vs. open |

## 2. Foundational theory libraries (durable, not volatile)

These are the perennial theoretical bases management theory builds on or against. Knowing
which conversation you are in is more important than any single citation.

- Organizational economics: transaction-cost economics, agency theory, property rights
- Behavioral foundations: bounded rationality, the behavioral theory of the firm, attention-based view
- Sociological foundations: institutional theory, resource dependence, organizational ecology, social network theory
- Strategy foundations: resource-based view, dynamic capabilities, knowledge-based view
- Micro-foundations: sensemaking, identity, cognition, emotion, paradox

current AOM style guide.

## 4. Conceptual figures (the only "figures" AMR uses)

AMR figures carry *theoretical* work: process models, 2x2 typologies, multi-level
frameworks, recursive/feedback loops. They are NOT data plots.

| Tool | Use |
|------|-----|
| Microsoft PowerPoint / Keynote | Most common for box-and-arrow process models; export to high-resolution image |
| draw.io (diagrams.net) | Free; clean orthogonal connectors for multi-stage process models |
| Lucidchart | Collaborative diagramming; good for typologies and 2x2 matrices |
| OmniGraffle (macOS) | Precise control for publication-quality conceptual models |
| TikZ / LaTeX | For authors who draft in LaTeX; precise typologies and process diagrams |

Design rule: every box must be a defined construct, every arrow a stated proposition or
mechanism. A figure that cannot be read off the propositions is decoration, not theory.

## 5. Argument-logic aids

Theory papers are evaluated on logical soundness. These help you stress-test the argument
before reviewers do.

- Argument-mapping tools (e.g., Rationale, MindMup, or a simple premise → premise → conclusion outline) to expose hidden assumptions
- Toulmin model (claim / grounds / warrant / backing / rebuttal) as a per-proposition checklist
- Counterfactual / disconfirming-case worksheets: for each proposition, name a case where it would NOT hold and check whether a boundary condition covers it
- Thought-experiment logs: write the "if construct X moved, then Y" chain in prose before formalizing the proposition

## 8. Cautions

1. **No data, ever.** If you find yourself wanting a "sample," "measure," or "test," the project belongs at AMJ/ASQ/SMJ, not AMR. AMR builds the theory those journals later test.
2. **Engage the home conversation.** Verify you have read the recent AMR/AMJ pieces on your construct; reviewers are usually drawn from exactly that stream.
3. **Verify limits.** Length, abstract, and formatting limits change — confirm on the current author page before you submit.
4. **Originality.** Run the manuscript through your institution's similarity check and confirm it has not been published or is not under review elsewhere, per the AOM Code of Ethics.
