# 定性路径（socs-qualitative）

## 触发时机

- 材料是田野民族志、深度访谈、扩展个案、历史比较
- 稿子读起来像"讲了个好故事"，但没分析抓手
- 审稿担心："材料够吗、编码透明吗、能不能信"

## 立场：材料厚度 + 分析深度 + 可信度

本刊接纳定性传统，但门槛不低于定量。**复述故事不是分析；从材料中提炼概念/机制、与理论往返，才是。**

## 方法路数（明确你站在哪一支）

| 路数 | 核心动作 | 产出 |
|------|----------|------|
| 田野民族志 | 长期在场、深描、参与观察 | 对社会过程的厚描述与机制 |
| 深度访谈 | 半结构、理论抽样、饱和 | 意义建构与类型学 |
| 扩展个案法 | 用个案对话/修正既有理论 | 理论的延伸与修正 |
| 扎根理论 | 开放/主轴/选择性编码 | 自下而上的概念与命题 |
| 历史比较 | 过程追踪、比较案例 | 机制与因果叙事 |

## 可信度（trustworthiness）四要件

- **材料厚度**：在场时长、访谈数、文档来源，足以支撑论断
- **编码透明**：如何从原始材料到概念，可被追溯（不是"我感觉")
- **反身性**：研究者位置、进入方式、对材料的影响有交代
- **饱和与负面案例**：是否到饱和，是否处理了不符合的个案

## 从材料到概念（本刊最看重）

- 不停在"他们说了什么"，要问"这揭示了什么社会机制"
- 用 `socs-concept-building` 提炼概念，用 `socs-theory-dialogue` 与理论往返
- 引用材料要**典型且足量**，避免孤证支撑大判断

## 自检清单

- [ ] 明确方法路数（民族志/访谈/扩展个案/扎根/历史比较）
- [ ] 材料厚度足以支撑论断（时长/样本/来源交代清楚）
- [ ] 从材料到概念的编码过程可追溯、透明
- [ ] 有反身性与负面案例处理
- [ ] 已从故事提炼出概念/机制，而非复述
- [ ] 匿名与伦理（化名、知情同意）已处理

## 反模式

- 故事复述，无概念无机制
- 用个别金句支撑普遍结论（孤证）
- 编码黑箱，无法追溯
- 把"新田野点"当贡献本身
- 忽略反身性与研究伦理

## 输出格式

```
【路数】民族志 / 访谈 / 扩展个案 / 扎根 / 历史比较
【材料厚度】在场<…> / 访谈 N=<…> / 文档<…>
【可信度】编码透明□ 反身性□ 饱和□ 负面案例□
【已提炼概念/机制】<…>（→ socs-concept-building）
【伦理】化名□ 知情同意□
【下一步】socs-concept-building / socs-mechanism-social-process
```
