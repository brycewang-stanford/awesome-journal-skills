# 摘要与关键词（socs-abstract-keywords）

## 触发时机

- 正文、问题意识、机制/概念已定，准备写摘要
- 摘要写成背景套话，或超字数
- 关键词随手凑，没体现问题意识与机制

## 本刊硬规范（详见 resources/journal-profile.md）

- **中文摘要不超过 200 字**
- **关键词 3–5 个**（中文）
- **英文摘要不超过 150 个单词**
- 中英文标题、摘要、关键词齐全

> 以官网最新《投稿指南》为准；字数是硬约束，别超。

## 摘要四要素（命题前置，不先铺背景）

1. **问题/争议**：本文要回答的社会学问题（1 句，别长篇背景）
2. **数据/方法**：CGSS/CFPS/CLDS 或田野/访谈，一句带过
3. **核心发现/机制**：经由什么社会过程，形成什么结构（重点）
4. **理论贡献**：提出什么概念/修正什么判断（落点）

## 写法对照

| 黑名单（背景腔） | 白名单（命题前置） |
|------------------|--------------------|
| "随着……日益重要，本文研究……" | "本文追问 <机制> 如何在 <条件> 下再生产 <不平等>" |
| "运用先进方法分析了数据" | "基于 CFPS，借助事件史模型……" |
| "丰富了相关研究" | "提出 <概念>，修正了 <理论判断>" |

## 关键词选取

- 体现：**问题域 + 理论概念 + 方法/数据**（如：教育不平等、文化资本、生命历程、CFPS）
- 3–5 个，避免过宽（"社会""中国"）或与标题完全重复
- 至少 1 个体现理论对话/原创概念

## 自检清单

- [ ] 中文摘要 ≤ 200 字（已计字）
- [ ] 关键词 3–5 个，含理论概念
- [ ] 英文摘要 ≤ 150 词，与中文一致
- [ ] 命题/问题前置，背景不超过 1 句
- [ ] 含机制与贡献，不止"研究了什么"
- [ ] 中英文标题/摘要/关键词齐全

## 反模式

- 摘要 2/3 在铺背景
- 只说做了什么，不说发现了什么机制
- 关键词全是宽泛大词或纯方法名
- 超 200 字还不自知

## 输出格式

```
【中文摘要】<…>（X 字，≤200）
【关键词】<…>；<…>；<…>（3–5）
【英文摘要】<…>（X words，≤150）
【命题前置】是 / 否
【含机制+贡献】是 / 否
【下一步】socs-submission
```
