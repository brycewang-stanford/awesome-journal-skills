# 学理文风（socs-style）

## 触发时机

- 通读觉得"像政策汇报"或"像方法报告"
- 结尾全是"政府应加强/完善/推进/深化"
- 方法描述长于社会学阐释
- 大词空转，机制不落地

## 目标语域：社会学的学理克制

本刊的好文风**两头都不靠**：既不是政策汇报腔（把研究写成对策建议），也不是纯技术腔（把社会学问题写成模型展示）。每个判断回扣社会过程，每个大词能兑现为机制与边界。

## 对照表（黑名单 → 白名单）

| 问题 | 黑名单 | 白名单 |
|------|--------|--------|
| 政策汇报腔 | "政府应加强、完善、推进、深化……" | "这一机制表明，<结构条件> 是再生产 <不平等> 的关键环节" |
| 纯技术腔 | "本文采用多层事件史模型，AIC 更优……" | "借助事件史模型，本文得以分离出 <社会过程>" |
| 空洞价值 | "具有重要的理论与现实意义" | "提出 <概念>，重排了对 <现象> 的既有理解" |
| 套话贡献 | "丰富了相关研究" | "修正了 <理论判断> 在 <条件> 下的适用性" |
| 大词空转 | "推动社会治理现代化" | 落到具体机制与行动者，不用不可兑现的大词 |
| 系数崇拜 | "X 显著为正（p<0.01)" | "X 对应约 <…> 的机会差距，意味着 <社会后果>" |

## 行文规范

- **一段一概括**：每段首句是概括句，审稿人读首句即可复原逻辑
- 概念全文一致，慎用未界定的大词与生造术语
- 谦逊与边界：经验研究说"提供了证据支持"，不说"证明了"
- 引用材料/系数后必须**回到社会学解释**，不让证据裸奔
- 方法段不长于机制与理论阐释段

## 自检清单

- [ ] "具有重要意义"类空话命中 = 0
- [ ] 无政策汇报四件套（加强/完善/推进/深化）
- [ ] 无纯炫技段（方法长度 ≤ 解释长度）
- [ ] 每段首句为概括句
- [ ] 系数/材料后都有社会学解读
- [ ] 大词都能兑现为机制/边界

## 反模式

- 通篇对策建议，把社会学写成政策报告
- 模型与拟合指标占据主场，社会过程缺席
- 用排比大词替代论证

## 输出格式

```
【空话命中】X 处：[...]
【政策腔】有 / 无    【技术腔】有 / 无
【段首概括率】X / 总段数
【证据裸奔】X 处（系数/材料无解读）
【大词兑现】到位 / 空转 [...]
【下一步】socs-abstract-keywords
```
