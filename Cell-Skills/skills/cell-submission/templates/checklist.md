# Cell Submission Preflight Checklist

> Copy this file, tick every box, and resolve all blocking items before upload.
> Targets reflect typical Cell Press guidelines — confirm exact caps in the current Author Guidelines.

## 1. Significance & narrative
- [ ] Complete, mechanistic story established (cell-fit rung ≥ 4; multiple converging lines)
- [ ] Single narrative arc: hypothesis → mechanism → significance (no parallel mini-stories)
- [ ] Causality shown (loss- and gain-of-function or equivalent perturbation pair)
- [ ] Title declarative, names the actor + effect

      char counts: 1) ___  2) ___  3) ___  4) ___
- [ ] eTOC / In Brief blurb: ~50 words, third person, lay language: ______ words
- [ ] Graphical Abstract: single panel ☐ clear flow ☐ minimal text ☐ RGB ☐ correct size ☐

## 3. Summary & writing
- [ ] Summary: single paragraph ≤ ~150 words: ______ words
- [ ] Summary names the mechanism (the *how*), not just the phenomenon
- [ ] ≥ 1 quantified headline result in Summary; no citations/subheadings
- [ ] Main text ~45,000 characters incl. legends: ______ chars
- [ ] Display items within ~7–8: ______
- [ ] Results have claim-style subheadings; Discussion interpretive (not a recap)

## 4. Figures & display items
- [ ] Sized to 85 / 114 / 174 mm; min font ~6–7 pt; RGB
- [ ] Data shown (points + n + replicate type); error bars defined (SD/SEM/CI)
- [ ] Scale bars on images; uncropped key blots + source data retained
- [ ] Colorblind-safe palette; meaning not by color alone
- [ ] Each legend: title sentence + per-panel + statistics (stand-alone)

## 5. STAR Methods
- [ ] Key Resources Table complete (all reagent/resource categories)
- [ ] Antibodies have Cat# + RRID; software has version + RRID:SCR/DOI
- [ ] Deposited Data rows list this paper's accessions/DOIs
- [ ] Resource Availability — all three subsections:
      ☐ Lead Contact  ☐ Materials Availability  ☐ Data and Code Availability
- [ ] Experimental Model and Subject Details (incl. IRB ☐ / IACUC ☐ / consent ☐ / authentication ☐)
- [ ] Method Details reproducible and KRT-linked
- [ ] Quantification and Statistical Analysis consolidated (test + n-definition + dispersion + software)

## 6. Data & code
- [ ] Data deposited; accessions / DOIs in hand: ______
- [ ] Code public + archived (Zenodo / Mendeley Data DOI): ______
- [ ] Three-bullet Data and Code Availability statement:
      ☐ DATA  ☐ CODE  ☐ ADDITIONAL
- [ ] Accessions mirrored into KRT "Deposited Data"
- [ ] Restricted/clinical data: controlled-access procedure stated

## 7. References
- [ ] Cell Press author–date in text "(Smith et al., 2021)"
- [ ] Reference list alphabetical by first author (NOT numbered, NOT by appearance)
- [ ] Full author lists; surname-first; journal-name style correct
- [ ] Every in-text (Author, year) resolves; a/b suffixes for same author/year

## 8. Required files & metadata
- [ ] Main text (Summary, Intro, Results, Discussion, STAR Methods, legends, references)
- [ ] Figures at required resolution + legends
- [ ] Graphical Abstract file
- [ ] Highlights + eTOC blurb fields/files
- [ ] Supplemental Information (figures/tables/videos + captions)
- [ ] Cover letter (conceptual-advance + completeness forward)
- [ ] Authors, affiliations, ORCIDs, corresponding author, Lead Contact
- [ ] CRediT author contributions
- [ ] Competing interests + funding statements
- [ ] Suggested / opposed reviewers (if used)

## 9. Final integrity sweep
- [ ] No over-claiming beyond the evidence (re-read Summary + Discussion)
- [ ] Every figure n / replicate type / error bar / test defined and consistent with QSA
- [ ] All accession numbers / DOIs live and correct
- [ ] Blinding convention matches the journal (Cell typically single-blind)
