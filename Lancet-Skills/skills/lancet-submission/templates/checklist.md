# The Lancet Submission Preflight Checklist

> Copy this file, tick every box, and resolve all blocking items before upload.
> Targets reflect typical Lancet / ICMJE guidance — confirm exact caps in the current Lancet Author Guidelines and ICMJE Recommendations.

## 1. Importance & framing
- [ ] Clinically/public-health important, globally relevant, practice-or-policy-changing (lancet-fit rung ≥ 3)
- [ ] Equity / global-health angle articulated where relevant
- [ ] Title declarative, specific, reflects design (e.g., "...: a randomised controlled trial")

## 2. Registration & design
- [ ] Trial registered **prospectively** (before first enrollment): registry ______ number ______
- [ ] Registration number in the **abstract** and Methods
- [ ] Protocol available; pre-specified **statistical analysis plan (SAP)** available
- [ ] Reported primary outcome matches the registered primary outcome (or change explained + dated)
- [ ] RCT: randomisation / allocation concealment / blinding / ITT described

## 3. Reporting guideline & flow diagram
- [ ] Guideline: ☐ CONSORT(+ext) ☐ STROBE ☐ PRISMA(+ext) ☐ SPIRIT ☐ other ______
- [ ] Completed checklist, mapped to page/line, submitted as a supplementary file
- [ ] **Flow diagram** present: ☐ CONSORT ☐ PRISMA ☐ N/A
- [ ] Flow-diagram numbers reconcile with Table 1 and analysis populations

## 4. Statistics
- [ ] Effect estimates with 95% CI; exact P values
- [ ] Absolute **and** relative effects reported (NNT/NNH where relevant)
- [ ] Pre-specified primary analysis; ITT primary; per-protocol sensitivity (esp. non-inferiority)
- [ ] Missing-data amount/pattern/handling + sensitivity analyses
- [ ] Multiplicity strategy for secondary endpoints stated
- [ ] Subgroups pre-specified; interaction tests (not within-subgroup P); cautious interpretation

## 5. Structured abstract
- [ ] ≤ ~300 words: ______ words
- [ ] Exact headings: Background / Methods / **Findings** / **Interpretation** / Funding
- [ ] Registration number in Methods
- [ ] Primary outcome with effect size + 95% CI
- [ ] Funder named (or "None")

## 6. Research in context panel
- [ ] Evidence before this study (systematic search: databases + terms + dates + language)
- [ ] Added value of this study (specific, non-promotional)
- [ ] Implications of all the available evidence (practice + policy + research)

## 7. Figures & tables
- [ ] Table 1 baseline characteristics (no baseline P values in an RCT)
- [ ] Kaplan–Meier with numbers at risk (if time-to-event)
- [ ] Forest plot for pre-specified subgroups / meta-analysis (with interaction p / heterogeneity)
- [ ] Map for multi-country / global-health study (if relevant)
- [ ] Confidence intervals shown on all effect estimates

## 8. Ethics, declarations & data
- [ ] Ethics committee approval(s) for all sites + informed consent + Declaration of Helsinki
- [ ] **Declaration of interests** for all authors (ICMJE form); explicit "none" where applicable
- [ ] ICMJE author contributions; who **verified the data** named
- [ ] **Role of the funding source** statement (funder role + data access + final responsibility to submit)
- [ ] **Data sharing statement** (ICMJE-compliant for trials)
- [ ] PPI statement; SAGER sex/gender reporting; race/ethnicity + PROGRESS-Plus where relevant

## 9. References & required files
- [ ] References within budget (~30 for an Article): ______
- [ ] All in-text citations resolve; correct reference style
- [ ] Main text (IMRaD + Research in context panel)
- [ ] Tables, figures + legends, appendix/supplement
- [ ] Reporting checklist; protocol + SAP; cover letter
- [ ] Authors, affiliations, ORCIDs, corresponding author

## 10. Final integrity sweep
- [ ] No over-claiming; causal language matches design; no superiority from a non-inferiority trial
- [ ] All abstract / flow-diagram / Table 1 / text numbers reconcile
- [ ] Registration number, ethics references, data-repository identifiers correct and live
- [ ] Headings are Findings / Interpretation (not Results / Conclusions)
